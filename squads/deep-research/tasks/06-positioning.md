# Task: Positioning

## Objective
Definir posicionamento único que diferencia das alternativas e ressoa com o público-alvo.

## Context
Posicionamento é a posição mental que você ocupa na mente do cliente. Não é o que você diz sobre você, é o que o cliente pensa quando ouve seu nome.

## Prompts de Apoio

### Para Positioning Statement
```
Crie um positioning statement para:

Target: [AUDIENCE]
Problem: [PROBLEM]
Category: [CATEGORY]
Product: [PRODUCT]

Use a fórmula:
"Para [target] que [need], [product] é [category] que [benefit].
Diferente de [alternatives], nós [unique differentiator]."

Crie 3 versões:
1. Funcional (foco em features)
2. Emocional (foco em feelings)
3. Aspiracional (foco em identity)
```

### Para Value Proposition
```
Desenvolva uma proposta de valor para [PRODUTO]:

1. Liste os 3 principais jobs-to-be-done do cliente
2. Liste as 3 maiores dores relacionadas
3. Liste os 3 maiores ganhos desejados
4. Mostre como o produto:
   - Resolve cada job
   - Alivia cada dor
   - Cria cada ganho

Finalize com uma frase de proposta de valor única.
```
