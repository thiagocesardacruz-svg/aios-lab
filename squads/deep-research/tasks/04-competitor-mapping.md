## Prompts de Apoio

### Para Competitive Analysis
```
Analise o competidor [NOME]:

1. Produto/Serviço
   - O que oferecem?
   - Qual o modelo de preço?
   - Principais features?

2. Posicionamento
   - Qual a mensagem principal?
   - Para quem se posicionam?
   - Qual o tom de comunicação?

3. Forças e Fraquezas
   - O que fazem muito bem?
   - Onde são fracos?
   - O que clientes reclamam? (reviews)

4. Oportunidades
   - Onde podemos ser diferentes?
   - Que necessidades não atendem?
```

### Para Messaging Analysis
```
Analise as mensagens de marketing dos competidores:
[LISTA DE COMPETIDORES COM URLS]

Para cada um, identifique:
1. Headline principal
2. Proposta de valor em uma frase
3. Principais provas/social proof
4. Call-to-action

Depois, analise:
- Padrões comuns (o que todos dizem)
- Gaps de mensagem (o que ninguém diz)
- Oportunidades de diferenciação
```
