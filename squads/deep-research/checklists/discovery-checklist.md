# Discovery Phase Checklist

## Phase Overview
**Objective:** Define the problem and understand the audience
**Duration:** Quick (2h) / Standard (4h) / Deep (1-2 days)

---

## Problem Definition

### Initial Capture
- [ ] Document the initial idea/opportunity
- [ ] Identify what motivated this exploration
- [ ] Define the ideal outcome

### Problem Statement
- [ ] Define WHO has the problem
- [ ] Define WHAT the problem is specifically
- [ ] Define WHEN/WHERE the problem occurs
- [ ] Define WHY it matters to solve
- [ ] Document current alternatives

### Job to Be Done
- [ ] Write JTBD statement: "When [situation], I want [motivation], so I can [result]"
- [ ] Validate JTBD makes sense

### Problem Validation
- [ ] Is the problem real (not imagined)?
- [ ] Do people pay to solve this today?
- [ ] Is the problem frequent enough?
- [ ] Is the pain strong enough to drive action?
- [ ] Is there a specific segment that suffers more?

### Scope
- [ ] Define what's IN scope
- [ ] Define what's OUT of scope
- [ ] List key assumptions

**Output:** `problem-statement.md`

---

## Audience Research

### Segmentation
- [ ] List all possible segments
- [ ] Evaluate potential of each
- [ ] Select primary segment
- [ ] Select secondary segment (if applicable)

### Primary Research (if doing Deep level)
- [ ] Conduct 5-10 interviews
- [ ] Run survey (if applicable)
- [ ] Observe behavior

### Secondary Research
- [ ] Analyze reviews and comments
- [ ] Study forums and communities
- [ ] Monitor social media
- [ ] Review industry reports

### Persona Development
- [ ] Define demographics
- [ ] Define psychographics
- [ ] Map functional pains
- [ ] Map emotional pains
- [ ] Map social pains
- [ ] Map functional desires
- [ ] Map emotional desires
- [ ] Map social desires

### Decision Journey
- [ ] Identify trigger
- [ ] Map awareness stage
- [ ] Map consideration stage
- [ ] Map decision stage
- [ ] Map usage stage
- [ ] Map loyalty factors

**Output:** `audience-personas.md`

---

## Phase Complete Criteria

- [ ] Problem is clearly articulated
- [ ] At least 1 persona is documented
- [ ] Pain points are evidence-based
- [ ] Desires are clearly mapped
- [ ] Ready to move to Analysis phase
