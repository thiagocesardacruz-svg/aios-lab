# Checklist: Copy Quality

## Uso
Validar qualidade do copy antes de publicar página de vendas.

---

## 1. Headline & Above The Fold

- [ ] Headline comunica benefício principal em < 10 palavras
- [ ] Sub-headline expande/qualifica a headline
- [ ] Proposta de valor clara em 5 segundos
- [ ] CTA visível sem scroll
- [ ] Prova social inicial presente

## 2. Clareza e Compreensão

- [ ] Linguagem adequada ao avatar (sem jargões)
- [ ] Frases curtas (< 20 palavras em média)
- [ ] Parágrafos curtos (< 4 linhas)
- [ ] Uma ideia por parágrafo
- [ ] Transições suaves entre seções

## 3. Persuasão e Emoção

- [ ] Conecta com dor/frustração do avatar
- [ ] Amplifica o problema antes da solução
- [ ] Benefícios > Features (ratio 3:1)
- [ ] Usa linguagem sensorial (ver, sentir, ouvir)
- [ ] Inclui storytelling de identificação

## 4. Uso de "Você"

- [ ] Ratio você/nós é > 4:1
- [ ] Evita "nós achamos" ou "nosso produto"
- [ ] Foca na transformação do leitor
- [ ] Usa segunda pessoa consistentemente

## 5. Power Words

- [ ] Palavras de urgência presentes
- [ ] Palavras de exclusividade usadas
- [ ] Palavras de garantia incluídas
- [ ] Palavras de facilidade aplicadas
- [ ] Distribuição balanceada ao longo do texto

## 6. Objeções

- [ ] Objeção de preço endereçada
- [ ] Objeção de tempo abordada
- [ ] Objeção de credibilidade tratada
- [ ] Objeção de aplicabilidade resolvida
- [ ] FAQ cobre dúvidas principais

## 7. Prova Social

- [ ] Mínimo 3 depoimentos
- [ ] Depoimentos com resultados específicos
- [ ] Variedade de perfis nos depoimentos
- [ ] Estatísticas de resultado incluídas
- [ ] Indicadores de autoridade presentes

## 8. Oferta

- [ ] Value stack claro e visual
- [ ] Valor de cada item especificado
- [ ] Economia destacada
- [ ] Garantia proeminente
- [ ] Urgência/escassez com justificativa real

## 9. CTAs

- [ ] CTA principal claro e action-oriented
- [ ] CTAs consistentes em toda a página
- [ ] Verbo de ação (não "clique aqui")
- [ ] Benefício implícito no CTA
- [ ] Mínimo 3 CTAs na página

## 10. Formatação

- [ ] Headers H2/H3 para cada seção
- [ ] Bullet points para listas
- [ ] Bold para pontos-chave
- [ ] Espaçamento adequado
- [ ] Escaneabilidade testada (skim test)
