# Checklist: Compliance & Legal

## Uso
Garantir que a página de vendas está em conformidade com regulamentações e boas práticas.

---

## 1. Veracidade das Alegações

### Resultados
- [ ] Resultados prometidos são realistas
- [ ] Casos de sucesso são verificáveis
- [ ] Disclaimer "resultados podem variar" presente
- [ ] Não promete resultados garantidos
- [ ] Timeframes são razoáveis

### Depoimentos
- [ ] Depoimentos são reais (não inventados)
- [ ] Permissão obtida para uso
- [ ] Resultados atípicos identificados
- [ ] Não usa fotos de banco de imagem como "clientes"

### Credenciais
- [ ] Certificações são válidas
- [ ] Experiência é verificável
- [ ] Títulos são precisos
- [ ] Afiliações são atuais

---

## 2. Proteção ao Consumidor

### Direito de Arrependimento
- [ ] Garantia de 7 dias mínimo (Brasil - CDC)
- [ ] Processo de reembolso claro
- [ ] Não cria obstáculos para devolução
- [ ] Email/canal de suporte visível

### Informações do Produto
- [ ] O que está incluído é claro
- [ ] Formato de entrega especificado
- [ ] Duração de acesso definida
- [ ] Requisitos técnicos listados (se houver)

### Preço
- [ ] Preço total visível antes da compra
- [ ] Parcelas incluem juros (se houver)
- [ ] Moeda especificada
- [ ] Sem taxas ocultas

---

## 3. Privacidade e Dados

### LGPD (Brasil)
- [ ] Política de privacidade presente
- [ ] Finalidade de coleta de dados clara
- [ ] Consentimento explícito para emails
- [ ] Opção de descadastro em emails
- [ ] Dados de contato do responsável

### Cookies
- [ ] Aviso de cookies (se aplicável)
- [ ] Opção de recusar cookies não-essenciais
- [ ] Lista de cookies usados disponível

---

## 4. Marketing Ético

### Urgência/Escassez
- [ ] Deadlines são reais (não fake)
- [ ] Contadores não resetam
- [ ] Limitações de vagas são verdadeiras
- [ ] Preços promocionais têm data real

### Comparações
- [ ] Não difama concorrentes
- [ ] Comparações são justas e verificáveis
- [ ] Não usa nomes de marca sem permissão

### Afiliados
- [ ] Divulgação de links afiliados (se houver)
- [ ] Transparência em parcerias
- [ ] Disclaimer de compensação

---

## 5. Acessibilidade

### Visual
- [ ] Contraste de cores adequado (4.5:1)
- [ ] Texto legível (16px mínimo)
- [ ] Não depende só de cor para informação
- [ ] Imagens têm alt text

### Navegação
- [ ] Navegável por teclado
- [ ] Focus states visíveis
- [ ] Estrutura de headings lógica
- [ ] Links descritivos

---

## 6. Plataformas de Anúncios

### Facebook/Meta
- [ ] Não viola políticas de saúde/finanças
- [ ] Não faz alegações de enriquecimento fácil
- [ ] Sem antes/depois proibidos
- [ ] Sem linguagem sensacionalista excessiva

### Google
- [ ] Página de destino relevante
- [ ] Não é "bridge page" vazia
- [ ] Informações de contato presentes
- [ ] HTTPS obrigatório

---

## 7. Pagamentos

### Segurança
- [ ] SSL certificado válido
- [ ] Gateway de pagamento confiável
- [ ] PCI compliance (se processa cartões)
- [ ] Não armazena dados de cartão

### Transparência
- [ ] Nome que aparece na fatura claro
- [ ] Política de cancelamento (se recorrente)
- [ ] Política de reembolso acessível
