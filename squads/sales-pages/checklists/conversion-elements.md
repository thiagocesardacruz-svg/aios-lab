# Checklist: Conversion Elements

## Uso
Validar presença de todos elementos essenciais para conversão.

---

## 1. Trust Elements

### Credibilidade
- [ ] Logo ou nome do criador visível
- [ ] Credenciais/experiência mencionadas
- [ ] Menções na mídia (se houver)
- [ ] Certificações/prêmios (se houver)
- [ ] Anos de experiência destacados

### Segurança
- [ ] Badge de pagamento seguro
- [ ] Ícone de cadeado/SSL
- [ ] Logos de cartões aceitos
- [ ] Política de privacidade linkada
- [ ] Termos de uso disponíveis

### Prova Social
- [ ] Número de clientes/alunos
- [ ] Avaliações com estrelas
- [ ] Depoimentos com foto
- [ ] Case studies
- [ ] Logos de empresas/clientes (B2B)

---

## 2. Urgency & Scarcity

### Urgência Temporal
- [ ] Countdown timer (se deadline real)
- [ ] Data limite clara
- [ ] "Disponível até" especificado
- [ ] Razão para deadline explicada

### Escassez de Quantidade
- [ ] Número de vagas limitado
- [ ] Contador de vagas restantes
- [ ] "Apenas X disponíveis"
- [ ] Justificativa da limitação

### Escassez de Bônus
- [ ] Bônus com expiração
- [ ] "Bônus removido após X"
- [ ] Indicador visual de bônus limitado

---

## 3. Risk Reversal

### Garantia
- [ ] Garantia claramente visível
- [ ] Duração especificada
- [ ] Condições claras (se houver)
- [ ] Processo de devolução simples
- [ ] Badge visual de garantia

### Suporte
- [ ] Canais de suporte listados
- [ ] Tempo de resposta mencionado
- [ ] FAQ para dúvidas comuns

---

## 4. CTA Optimization

### Design
- [ ] Cor contrastante com a página
- [ ] Tamanho adequado (mobile-friendly)
- [ ] Espaço branco ao redor
- [ ] Sombra/elevação para destaque

### Copy
- [ ] Verbo de ação no início
- [ ] Benefício implícito ou explícito
- [ ] Urgência sutil (se apropriado)
- [ ] Sem palavras negativas

### Posicionamento
- [ ] CTA above the fold
- [ ] CTA após cada seção principal
- [ ] CTA final proeminente
- [ ] Sticky CTA opcional (mobile)

---

## 5. Page Structure

### Above The Fold
- [ ] Headline impactante
- [ ] Subheadline de suporte
- [ ] Visual relevante
- [ ] CTA visível
- [ ] Micro-prova social

### Flow
- [ ] Progressão lógica de seções
- [ ] Problema antes da solução
- [ ] Prova social distribuída
- [ ] Oferta após valor estabelecido
- [ ] Múltiplos pontos de conversão

### Mobile
- [ ] Responsivo testado
- [ ] Touch targets adequados (44px+)
- [ ] Texto legível sem zoom
- [ ] Imagens otimizadas
- [ ] Carregamento < 3 segundos

---

## 6. Friction Reduction

### Formulário
- [ ] Campos mínimos necessários
- [ ] Labels claras
- [ ] Erros inline
- [ ] Autocomplete habilitado
- [ ] Opção de checkout com 1 clique (se possível)

### Navegação
- [ ] Sem menu distrativo
- [ ] Links externos em nova aba
- [ ] Scroll suave
- [ ] Âncoras para seções longas
