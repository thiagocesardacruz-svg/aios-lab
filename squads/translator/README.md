# Translator Squad

> Translation & Localization

## Overview

The Translator Squad handles all multilingual needs for Travel Tech Digital. Responsible for:

- Content translation
- Cultural localization
- Terminology consistency
- Glossary management
- Translation quality review

## Agents

| Agent | Role | Focus |
|-------|------|-------|
| **Lingua** (translation-lead) | Lead | Coordination & standards |
| **Translators** | Translators | Language specialists |
| **Local** (localization-specialist) | Localization | Cultural adaptation |

## Languages

### Primary
- English (EN)
- Portuguese Brazil (PT-BR)
- Spanish (ES)

### Secondary
- French (FR)
- German (DE)
- Italian (IT)
- Arabic (AR)

## Commands

| Command | Description |
|---------|-------------|
| `/translate/localize-content` | Translate and localize |
| `/translate/translate-batch` | Batch translation |
| `/translate/review` | Review quality |
| `/translate/glossary-update` | Update glossary |

## Process

```
English Source → Translation → Review → Localization → Final QA
```

## Guidelines

1. **Never word-for-word** - Translate meaning, not words
2. **Cultural fit** - Adapt idioms, references, examples
3. **Glossary first** - Check terminology before translating
4. **Native review** - Always have native speaker review
