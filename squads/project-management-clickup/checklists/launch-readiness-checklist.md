# Checklist: Launch Readiness

> Checklist de prontidão para lançamentos de infoprodutos

---

## T-30 dias (Pré-Pré-Lançamento)

### Estratégia
- [ ] Meta de faturamento definida
- [ ] Meta de unidades vendidas calculada
- [ ] Avatar e dores mapeados
- [ ] Oferta principal estruturada
- [ ] Order bump definido
- [ ] Upsells/downsells planejados
- [ ] Preço e condições de pagamento
- [ ] Datas do lançamento confirmadas

### Equipe
- [ ] Time de lançamento definido
- [ ] Responsabilidades distribuídas
- [ ] Canais de comunicação configurados
- [ ] Reuniões de acompanhamento agendadas

### Infraestrutura Inicial
- [ ] Workspace no ClickUp criado
- [ ] Lista de tarefas importada
- [ ] Automações básicas configuradas
- [ ] Dashboard de métricas preparado

---

## T-14 dias (Pré-Lançamento)

### Conteúdo de Pré-Lançamento
- [ ] CPL 1 gravado e editado
- [ ] CPL 2 gravado e editado
- [ ] CPL 3 gravado e editado
- [ ] Páginas de CPL configuradas
- [ ] Emails de CPL escritos e revisados
- [ ] Sequência de emails agendada

### Captação
- [ ] Página de captura no ar
- [ ] Thank you page configurada
- [ ] Automação de boas-vindas ativa
- [ ] Pixel de conversão instalado
- [ ] Campanhas de tráfego aprovadas
- [ ] Budget de captação definido

### Testes
- [ ] Formulário de captura testado
- [ ] Emails chegando corretamente
- [ ] Tags sendo aplicadas
- [ ] Integração com CRM funcionando

---

## T-7 dias (Últimos Ajustes)

### Página de Vendas
- [ ] Copy finalizado e revisado
- [ ] Design aprovado
- [ ] Responsivo (mobile/desktop)
- [ ] Depoimentos inseridos
- [ ] FAQ completo
- [ ] Botões de CTA funcionando
- [ ] Escassez/urgência configurada
- [ ] Tempo de carregamento < 3s

### Checkout
- [ ] Produto cadastrado na plataforma
- [ ] Preço correto
- [ ] Parcelamento configurado
- [ ] Cupons de desconto (se houver)
- [ ] Pixel de purchase instalado
- [ ] Email de confirmação configurado
- [ ] Acesso automático configurado

### Testes de Checkout
- [ ] Compra teste com cartão
- [ ] Compra teste com PIX
- [ ] Compra teste com boleto
- [ ] Acesso liberado automaticamente
- [ ] Email de boas-vindas enviado

---

## T-1 dia (Véspera)

### Validação Final
- [ ] Todos os links funcionando
- [ ] Página de vendas no ar
- [ ] Checkout funcional
- [ ] Emails agendados confirmados
- [ ] Time de vendas briefado
- [ ] Suporte com FAQ atualizado
- [ ] WhatsApp de atendimento configurado

### Contingência
- [ ] Contato da plataforma de pagamento
- [ ] Backup da página de vendas
- [ ] Plano B para problemas técnicos
- [ ] Decisor disponível para emergências

### Comunicação
- [ ] Mensagem de abertura pronta
- [ ] Posts de redes sociais agendados
- [ ] Stories preparados
- [ ] Live de abertura configurada

---

## D-Day (Abertura)

### Hora H
- [ ] Email de abertura enviado
- [ ] Posts publicados
- [ ] Stories no ar
- [ ] Live iniciada (se houver)
- [ ] WhatsApp pronto para atender
- [ ] Dashboard de vendas aberto

### Monitoramento (a cada 2h)
- [ ] Verificar número de vendas
- [ ] Verificar taxa de abertura de emails
- [ ] Monitorar comentários/DMs
- [ ] Verificar se checkout está funcionando
- [ ] Atualizar time sobre progresso

---

## Durante o Lançamento

### Diário
- [ ] Enviar email do dia
- [ ] Publicar conteúdo planejado
- [ ] Responder comentários e DMs
- [ ] Atender leads no WhatsApp
- [ ] Atualizar dashboard de métricas
- [ ] Briefing rápido com time (15min)

### Tratamento de Problemas
- [ ] Boletos não compensados → seguir script de cobrança
- [ ] Dúvidas frequentes → atualizar FAQ
- [ ] Objeções → usar respostas do playbook
- [ ] Bugs → escalar para tech imediatamente

---

## Fechamento do Carrinho

### Últimas 24h
- [ ] Email de urgência enviado
- [ ] Contagem regressiva ativa
- [ ] Stories de última chamada
- [ ] Atendimento reforçado

### Última hora
- [ ] Email de última chance
- [ ] Monitorar vendas em tempo real
- [ ] Preparar para fechar checkout

### Após fechamento
- [ ] Desativar página de vendas
- [ ] Enviar email de encerramento
- [ ] Iniciar atendimento pós-venda
- [ ] Consolidar números

---

## Pós-Lançamento

### Imediato (D+1)
- [ ] Email de boas-vindas aos compradores
- [ ] Adicionar ao grupo VIP
- [ ] Iniciar onboarding
- [ ] Tratar pendências de pagamento

### Primeira Semana (D+7)
- [ ] Pesquisa de satisfação enviada
- [ ] Métricas consolidadas
- [ ] Relatório inicial gerado
- [ ] Recuperação de boletos

### Fechamento (D+14)
- [ ] Relatório final do lançamento
- [ ] Retrospectiva com time
- [ ] Documentação de aprendizados
- [ ] Planejamento de melhorias
