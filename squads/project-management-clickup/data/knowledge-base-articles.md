# Knowledge Base Articles Index - Support Operations

## Purpose

Index of common support topics and resolution templates for the support-operations-specialist. Each article provides a quick-reference answer and ClickUp template linkage.

---

## Category: Acesso ao Produto

### ART-001: Nao consigo acessar o curso
**Tags:** acesso, hotmart, login
**SLA:** 4h
**Resolution:**
1. Verificar se compra foi confirmada no Hotmart
2. Verificar email cadastrado na compra vs email de login
3. Enviar link de recuperacao de senha: Hotmart Club > Esqueci minha senha
4. Se persistir: verificar se pagamento foi aprovado (boleto pode levar 3 dias uteis)

### ART-002: Perdi acesso apos trocar email
**Tags:** acesso, email, hotmart
**SLA:** 24h
**Resolution:**
1. Confirmar identidade do aluno (email original + nome completo)
2. Acessar Hotmart painel > Alunos > buscar por email original
3. Atualizar email se solicitado pelo aluno
4. Enviar novo link de acesso

### ART-003: Acesso ao grupo VIP (WhatsApp/Telegram)
**Tags:** acesso, grupo, whatsapp
**SLA:** 24h
**Resolution:**
1. Verificar se compra inclui acesso ao grupo
2. Confirmar compra no Hotmart
3. Enviar link do grupo via email ou WhatsApp

## Category: Pagamento e Financeiro

### ART-010: Solicitar reembolso
**Tags:** reembolso, pagamento, financeiro
**SLA:** 24h (requer aprovacao)
**Resolution:**
1. Verificar prazo de garantia (7 dias para digital)
2. Se dentro do prazo: processar via Hotmart > Reembolsos
3. Se fora do prazo: avaliar caso (encaminhar para aprovacao)
4. Confirmar estorno ao aluno (prazo: 7-15 dias uteis)

### ART-011: Boleto nao compensou
**Tags:** boleto, pagamento
**SLA:** 24h
**Resolution:**
1. Verificar status no Hotmart (aguardando pagamento)
2. Confirmar data de vencimento
3. Se vencido: gerar novo boleto ou sugerir cartao/pix
4. Boletos levam ate 3 dias uteis para compensar

### ART-012: Parcelamento / Segunda via
**Tags:** parcela, pagamento
**SLA:** 24h
**Resolution:**
1. Verificar detalhes da assinatura/parcelamento no Hotmart
2. Se cartao recusado: orientar troca de cartao
3. Enviar link de segunda via se aplicavel

## Category: Conteudo e Produto

### ART-020: Material complementar / Downloads
**Tags:** material, download, conteudo
**SLA:** 24h
**Resolution:**
1. Indicar secao de materiais dentro do curso
2. Verificar se modulo esta liberado (lancamento faseado)
3. Se material esta faltando: reportar ao time de conteudo

### ART-021: Duvida sobre conteudo do curso
**Tags:** duvida, conteudo, tecnica
**SLA:** 72h
**Resolution:**
1. Se pergunta tecnica sobre fisioterapia: direcionar para FAQ do curso
2. Se duvida sobre metodologia: indicar modulo especifico
3. Se precisa atencao personalizada: agendar monitoria (se disponivel)

## Category: Tecnico

### ART-030: Video nao carrega
**Tags:** tecnico, video, player
**SLA:** 4h
**Resolution:**
1. Orientar limpar cache do navegador
2. Testar em navegador diferente (Chrome recomendado)
3. Verificar conexao de internet
4. Se persistir: reportar ao Hotmart suporte tecnico

### ART-031: App Hotmart - problemas
**Tags:** tecnico, app, mobile
**SLA:** 24h
**Resolution:**
1. Atualizar app para ultima versao
2. Limpar cache do app
3. Deslogar e logar novamente
4. Se persistir: usar navegador mobile como alternativa
