# Daily Report Template

## Meta
- **ID:** daily-report-tmpl
- **Squad:** ops
- **Used by:** generate-status-report, daily-standup
- **Output format:** md
- **Version:** 1.0.0

## Variables
| Variable | Description | Required | Default |
|----------|-------------|----------|---------|
| `{{report_date}}` | Date of the report (YYYY-MM-DD) | yes | none |
| `{{os_created}}` | Number of OS created today | yes | none |
| `{{os_completed}}` | Number of OS completed today | yes | none |
| `{{os_in_progress}}` | Number of OS currently in progress | yes | none |
| `{{os_blocked}}` | Number of OS currently blocked | yes | none |
| `{{daily_cost}}` | Total cost for the day | yes | none |
| `{{cost_by_squad}}` | Breakdown of cost per squad | yes | none |
| `{{blockers_list}}` | List of current blockers | no | "None" |
| `{{escalations}}` | Escalations triggered today | no | "None" |
| `{{sla_breaches}}` | SLA breaches today | no | "None" |
