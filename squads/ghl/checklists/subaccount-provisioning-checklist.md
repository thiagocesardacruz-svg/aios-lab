# Subaccount Provisioning Checklist

## Pre-provisioning

- [ ] Client data collected (business name, address, timezone)
- [ ] Niche identified and documented
- [ ] Snapshot selected for niche
- [ ] PIT token with required scopes available

## Provisioning

- [ ] `POST /locations` executed with snapshotId
- [ ] Location ID received and stored
- [ ] Snapshot loading confirmed
- [ ] Basic settings verified (timezone, business name)

## Post-provisioning

- [ ] Custom fields created by CRM Structuralist
- [ ] Pipeline configured
- [ ] Funnel deployed with domain
- [ ] Email templates created
- [ ] Workflows activated
- [ ] Webhook integrations tested

## Handover

- [ ] All component IDs documented
- [ ] Handover contract created
- [ ] Client notified of setup completion
