# Finance Squad

> Financial Viability, Pricing & Budget Control

## Overview

The Finance Squad ensures financial health of Travel Tech Digital. Responsible for:

- Financial planning and analysis
- Unit economics and KPIs
- Pricing strategy
- Budget management
- Cost control

## Agents

| Agent | Role | Focus |
|-------|------|-------|
| **Sterling** (finance-lead) | Finance Lead | Strategy & control |
| **Morgan** (financial-analyst) | Analyst | Unit economics & KPIs |
| **Price** (pricing-strategist) | Pricing | Tiers & margins |

## Commands

| Command | Description |
|---------|-------------|
| `/finance/report` | Financial report |
| `/finance/kpi` | KPI analysis |
| `/finance/pricing-review` | Review pricing strategy |
| `/finance/budget-check` | Budget check and alerts |

## Key Metrics

| Metric | Target |
|--------|--------|
| CAC | < €50 |
| LTV | > €150 |
| LTV:CAC | > 3:1 |
| Payback | < 6 months |
| Gross Margin | > 70% |
