# QA Squad

> Quality Assurance, Validation & Audits

## Overview

The QA Squad maintains quality standards across Travel Tech Digital. Responsible for:

- Quality gates before release
- Content review and validation
- Process audits
- Checklist management
- Continuous improvement

## Agents

| Agent | Role | Focus |
|-------|------|-------|
| **Shield** (qa-lead) | QA Lead | Gates & standards |
| **Echo** (content-reviewer) | Reviewer | Content quality |
| **Audit** (process-auditor) | Auditor | Process validation |

## Commands

| Command | Description |
|---------|-------------|
| `/qa/release-gate` | Final validation before deploy |
| `/qa/review-content` | Review content quality |
| `/qa/validate-output` | Validate against DoD |
| `/qa/audit-process` | Audit execution process |
| `/qa/run-checklist` | Run specific checklist |

## Quality Gate Flow

```
Created → Self-Review → Peer Review → QA Gate → Release
```

## When to Involve QA

- Before any external publication
- Before production deployment
- When claims need verification
- For process improvements
- After incidents (post-mortem)
