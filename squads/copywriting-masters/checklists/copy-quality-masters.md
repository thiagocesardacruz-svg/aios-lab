## Headline Quality (John Caples + Eugene Schwartz)

### Caples' Tested Principles
- [ ] Contains self-interest/benefit for reader
- [ ] Announces news or something new
- [ ] Arouses curiosity (but delivers)
- [ ] Offers quick, easy way
- [ ] Specific, not vague

### Schwartz Awareness Match
- [ ] Headline matches audience awareness level
- [ ] Unaware: Educates first
- [ ] Problem Aware: Agitates pain
- [ ] Solution Aware: Introduces your solution
- [ ] Product Aware: Differentiates
- [ ] Most Aware: Makes offer

### Headline Tests
- [ ] Passes "So What?" test
- [ ] Could work as standalone ad
- [ ] Specific numbers/results when possible
- [ ] Emotional trigger present
- [ ] No weak words (maybe, might, could)

## Lead Quality (Gary Halbert + Dan Kennedy)

### Halbert's A-Pile Test
- [ ] Feels personal, not mass-produced
- [ ] Would survive "A-pile" sort (not thrown away)
- [ ] Opens with reader's world, not yours
- [ ] Creates immediate connection
- [ ] No corporate-speak

### Kennedy's No B.S. Standards
- [ ] Gets to point quickly
- [ ] No wasted words
- [ ] Clear promise or hook
- [ ] Qualifies reader in/out
- [ ] Creates urgency naturally

### First 50 Words
- [ ] Capture attention immediately
- [ ] Promise or intrigue established
- [ ] Reader compelled to continue
- [ ] Voice/personality evident
- [ ] No "I" in first sentence (ideally)

## Body Copy Quality (Joe Sugarman + Robert Collier)

### Sugarman's Slippery Slide
- [ ] Each sentence compels reading the next
- [ ] No stopping points until CTA
- [ ] Curiosity maintained throughout
- [ ] Subheads create entry points
- [ ] Bucket brigades used ("But here's the thing...")

### Collier's Desire Formula
- [ ] Enters conversation in reader's mind
- [ ] Builds desire before revealing product
- [ ] Paints vivid picture of result
- [ ] Uses reader's own words/language
- [ ] Emotional, not just logical

### Psychological Triggers (Sugarman)
- [ ] Authority established
- [ ] Consistency/commitment used
- [ ] Reason why provided
- [ ] Specificity adds credibility
- [ ] Storytelling employed

---

## Persuasion Principles (Robert Cialdini)

### 6 Principles Applied
- [ ] **Reciprocity**: Value given before asking
- [ ] **Commitment**: Small yeses before big ask
- [ ] **Social Proof**: Others have done/bought
- [ ] **Authority**: Expertise demonstrated
- [ ] **Liking**: Personality/connection built
- [ ] **Scarcity**: Genuine limitation exists

### Ethical Standards
- [ ] Claims are truthful and provable
- [ ] Scarcity is genuine, not fabricated
- [ ] Social proof is real
- [ ] No manipulation, only persuasion

## Offer Structure (Russell Brunson + Alex Hormozi)

### Brunson's Value Stack
- [ ] Main offer clearly stated
- [ ] Bonuses add genuine value
- [ ] Price anchoring used
- [ ] Value to price ratio is obvious (10:1)
- [ ] Stack builds logically

### Hormozi's Value Equation
```
Value = (Dream Outcome × Perceived Likelihood)
        ────────────────────────────────────
        (Time Delay × Effort & Sacrifice)
```
- [ ] Dream outcome clearly articulated
- [ ] Likelihood of success increased
- [ ] Time to results minimized
- [ ] Effort required reduced

### Guarantee
- [ ] Risk reversal present
- [ ] Guarantee is bold, specific
- [ ] Terms are clear
- [ ] Builds confidence

## Email Quality (Ben Settle + André Chaperon)

### Settle's Infotainment Standards
- [ ] Entertaining AND informative
- [ ] Personality shines through
- [ ] Could be read for enjoyment alone
- [ ] Not just "content"
- [ ] Consistent voice

### Chaperon's Sequence Standards
- [ ] Opens loop that must be closed
- [ ] Story arc across emails
- [ ] Each email stands alone AND connects
- [ ] Relationship built, not just selling
- [ ] Anticipation created for next email

### Email Specifics
- [ ] Subject line creates curiosity
- [ ] First line hooks
- [ ] One main idea per email
- [ ] Clear CTA
- [ ] PS used strategically

## VSL Quality (Jon Benson + Stefan Georgi)

### Benson's VSL Structure
- [ ] Pattern interrupt opening
- [ ] Problem agitated fully
- [ ] Story creates connection
- [ ] Solution introduced at right moment
- [ ] Proof overwhelms skepticism
- [ ] Offer is irresistible
- [ ] Close is strong

### Georgi's RMBC Method
- [ ] **R**esearch evident in copy
- [ ] **M**echanism explained (why this works)
- [ ] **B**rief is followed
- [ ] **C**opy matches market sophistication

### Pacing
- [ ] Holds attention for full length
- [ ] No dull sections
- [ ] Builds to climax at offer
- [ ] Emotional peaks and valleys

## Launch Copy (Jeff Walker + Frank Kern)

### Walker's PLF Elements
- [ ] Pre-launch content builds anticipation
- [ ] Objections handled before launch
- [ ] Social proof built during pre-launch
- [ ] Scarcity is genuine (cart close)
- [ ] Story arc across launch

### Kern's Mass Control
- [ ] Intent-based positioning
- [ ] Goodwill before offer
- [ ] Personality-driven
- [ ] Creates movement/tribe
- [ ] Quick cash option available

## SEO Content (Brian Dean + Ann Handley)

### Dean's Standards
- [ ] 10x better than competition
- [ ] Comprehensive coverage
- [ ] Actionable and specific
- [ ] Linkable/shareable
- [ ] Optimized but natural

### Handley's Standards
- [ ] Useful to reader
- [ ] Clear and concise
- [ ] Empathetic tone
- [ ] Personality present
- [ ] Story when appropriate

## Final Quality Gate

### Overall Copy Health
- [ ] Single, clear message
- [ ] One reader in mind
- [ ] Benefits outweigh features (3:1)
- [ ] Proof supports all claims
- [ ] CTA is obvious and singular
- [ ] Would the master approve?

### Voice Consistency
- [ ] Tone consistent throughout
- [ ] Style matches selected master(s)
- [ ] No jarring shifts
- [ ] Personality maintained

### Technical Quality
- [ ] No typos or grammar errors
- [ ] Formatting aids readability
- [ ] Appropriate length for medium
- [ ] Mobile-friendly if applicable

---

## Master Approval Simulation

Before final delivery, ask:

> "Would [Master Name] be proud to put their name on this?"

If not, identify what's missing and revise.
