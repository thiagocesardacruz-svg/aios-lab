# Copywriting Masters Squad

> 22 Lendas do Copywriting em um Squad de Elite

## Overview

Squad de elite com os 22 maiores copywriters do mundo. Cada agente incorpora frameworks documentados, processos step-by-step e artefatos mentais extraídos de mestres reais do copywriting e persuasão.

**Domain:** Copywriting Excellence
**Version:** 1.1.0
**Total Components:** 40

## Como Funciona

```
Você → Copy Chief (Maverick) → Analisa missão → Roteia para especialista(s)
                                     ↓
                              Especialista executa
                                     ↓
                              Entrega copy no estilo do mestre
```

O **Copy Chief (Maverick)** é o ponto de entrada. Ele analisa sua missão e roteia para o(s) especialista(s) mais adequado(s).

## Frameworks dos Mestres

### Eugene Schwartz - 5 Níveis de Consciência
```
Unaware → Problem Aware → Solution Aware → Product Aware → Most Aware
```

### Robert Cialdini - 6 Princípios da Persuasão
1. Reciprocidade
2. Compromisso/Consistência
3. Prova Social
4. Autoridade
5. Escassez
6. Afeição

### André Chaperon - Soap Opera Sequence
```
Email 1: Set the Stage
Email 2: High Drama
Email 3: Epiphany
Email 4: Hidden Benefits
Email 5: Urgency & CTA
```

### Jeff Walker - Product Launch Formula
```
Pre-Pre-Launch → Pre-Launch → Launch → Post-Launch
    (Seed)        (Content)   (Cart)    (Debrief)
```

### Russell Brunson - Perfect Webinar
```
Introduction → Story → Content (3 Secrets) → Stack → Close
```

### Jon Benson - VSL Structure
```
Pattern Interrupt → Problem → Agitate → Story → Solution → Proof → Offer → Close
```

## Diferença: Copywriting Masters vs Sales Pages

| Aspecto | Copywriting Masters | Sales Pages |
|---------|---------------------|-------------|
| **Abordagem** | Consultoria de mestres | Workflow de produção |
| **Uso típico** | "Escreva como Dan Kennedy" | "Crie uma sales page" |
| **Output** | Copy no estilo do mestre | Página completa |
| **Profundidade** | Frameworks específicos | Processo estruturado |
| **Quantidade de agentes** | 23 especialistas | 3 agentes |

**São complementares:** Use Copywriting Masters para expertise profunda, Sales Pages para execução estruturada.

## Quality Standards

### Copy Review Checklist
- [ ] Headline passa no "So What?" test
- [ ] Lead captura atenção em 5 segundos
- [ ] Benefits > Features (ratio 3:1)
- [ ] Prova social específica e crível
- [ ] Urgência genuína (não fabricada)
- [ ] CTA claro e singular
- [ ] Voice consistente do início ao fim
