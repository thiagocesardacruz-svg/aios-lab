## Principios Universais
1. Copy existe para VENDER, nao para impressionar
2. Pesquisa ANTES de escrever (sempre)
3. Clareza > Criatividade
4. Beneficios > Features
5. Especifico > Generico
6. Uma ideia por vez
7. Escreva para UMA pessoa, nao para "audiencia"

## Regras de Qualidade
- Toda peca deve passar no "So What?" test (Carlton)
- Toda peca deve ter pelo menos 3 dos 7 principios de Cialdini
- Headlines devem ser testaveis (minimo 3 variacoes)
- Copy longo e permitido desde que cada frase justifique sua existencia (Sugarman Axiom 2)
- Proof elements sao obrigatorios (minimo 2 tipos diferentes)

## Formato de Entrega
- Headlines: sempre entregar 3+ opcoes rankeadas
- Emails: sempre incluir subject line + preview text
- Sales pages: entregar por secoes (Hero, Problem, Solution, Proof, Offer, CTA)
- VSLs: entregar com contagem de slides e timing estimado

## Review Obrigatorio
- Quick review: Carlton "So What?" test + Sugarman Slippery Slide check
- Standard review: + Wiebe Seven Sweeps
- Deep review: + Cialdini Persuasion Audit + Carlton SWS 17-point
