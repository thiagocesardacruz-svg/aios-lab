Every process step MUST have these 8 fields:

```markdown

|-------|-------|
| **Description** | What this step does |

| **Automation Potential** | HIGH/MEDIUM/LOW |
| **Executor** | Worker/Agent/Hybrid/Human |
| **Precondition** | What must be true before this step |
| **Output** | What this step produces |
| **Guardrails** | Constraints and safety checks |
| **Decision Rule** | When/how to apply this step |
```
