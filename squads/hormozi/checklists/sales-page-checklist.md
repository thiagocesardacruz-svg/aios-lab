## Pre-Flight Checklist

Before building the sales page, verify:

### Offer Documentation
- [ ] Product/service clearly defined
- [ ] Pricing finalized
- [ ] Value stack components listed with values
- [ ] Bonuses defined with values
- [ ] Guarantee policy approved

### Avatar Preparation
- [ ] Ideal customer profile documented
- [ ] Pain points identified (top 5)
- [ ] Desired outcomes identified
- [ ] Objections documented (top 5)
- [ ] Demographics for proof matching

### Proof Library
- [ ] Testimonials collected (minimum 3)
- [ ] Case studies available
- [ ] Result metrics documented
- [ ] Credibility elements (credentials, media)
- [ ] Proof organized by demographic

### Technical Setup
- [ ] Page builder ready
- [ ] Analytics tracking configured
- [ ] Heatmap tool installed (Hotjar/Clarity)
- [ ] A/B testing capability ready
- [ ] SSL certificate active

## SECTION A: Above the Fold (0-600px)
### Os 3 Segundos Criticos

> **Target Metric:** >40% scroll rate
> **Critical Rule:** These 5 elements decide stay or leave

---

### Element 1: Headline Principal (O Hook Magnetico)

**Value Equation Target:** Dream Outcome (MAXIMIZE)

**Formula Check:**
- [ ] Format: "Como [RESULTADO] em [TIMEFRAME] Sem [DOR #1] ou [DOR #2]"
- [ ] Contains specific result (not vague)
- [ ] Contains timeframe (days, weeks, months)
- [ ] Contains pain avoidance (at least one)

**Quality Check:**
- [ ] NOT feature-focused (e.g., "Curso de X")
- [ ] NOT generic benefit (e.g., "Aprenda X")
- [ ] IS outcome-specific with obstacles addressed

**Example Quality Spectrum:**
| Level | Example |
|-------|---------|
| RUIM | "Curso de Marketing Digital" |
| BOM | "Aprenda Marketing Digital" |
| MATADOR | "Como Conseguir 30 Clientes High-Ticket em 90 Dias Sem Gastar Fortuna em Ads ou Virar Escravo do Instagram" |

**A/B Test Ready:**
- [ ] Pain variant created
- [ ] Desire variant created
- [ ] Both variants follow formula

---

### Element 2: Subheadline (A Promessa Expandida)

**Value Equation Target:** Perceived Likelihood (MAXIMIZE)

**Template Check:**
```
"O [MECANISMO UNICO] usado por [NUMERO] [AVATAR ESPECIFICO]
para [RESULTADO COMPROVADO] - Agora disponivel para [QUALIFICACAO]"
```

**Element Verification:**
- [ ] Unique mechanism named
- [ ] Specific number of users/customers
- [ ] Avatar clearly identified
- [ ] Result stated (matching headline)
- [ ] Qualification included

**Positioning:**
- [ ] Directly below headline
- [ ] Font size: 60% of headline
- [ ] Same visual hierarchy

---

### Element 3: CTA Principal #1 (O Botao Impossivel de Ignorar)

**Value Equation Target:** Effort & Sacrifice (MINIMIZE)

**Technical Specifications:**
- [ ] Color: Minimum 3:1 contrast with background
- [ ] Size: Minimum 200px width (mobile)
- [ ] Position: Prominent, above fold

**Text Quality:**
- [ ] Benefit-oriented (NOT action-oriented)
- [ ] NOT "Comprar Agora" or "Buy Now"
- [ ] States outcome they want

**Quality Spectrum:**
| Level | Example |
|-------|---------|
| RUIM | "Comprar Agora" |
| BOM | "Quero Meu Acesso" |
| MATADOR | "Sim! Quero 30 Clientes em 90 Dias ->" |

**Psychological Hack:**
- [ ] Arrow (->) included in button text
- [ ] CTR boost expected: 12-26%

---

### Element 4: Prova Social Imediata (Trust Badge Instantaneo)

**Value Equation Target:** Perceived Likelihood (MAXIMIZE)

**Format Verification:**
```
⭐⭐⭐⭐⭐ [NUMBER]+ [USERS] | [SATISFACTION]% | [GUARANTEE/RESULT]
```

**Elements Present:**
- [ ] Star rating (5 stars)
- [ ] User/customer count (real number)
- [ ] Satisfaction percentage
- [ ] Result timeline or guarantee

**Example:**
"⭐⭐⭐⭐⭐ 2.847+ Alunos | 98% Satisfacao | Resultados em 14 Dias ou Dinheiro de Volta"

**Position:**
- [ ] Below CTA OR integrated in header
- [ ] Visible without scrolling
- [ ] Numbers are REAL and verifiable

---

### Element 5: Urgencia/Escassez Indicator

**Value Equation Target:** Loss Aversion (AMPLIFY)

**Type Selection (choose one):**
- [ ] Countdown Timer: "Oferta Expira em 23:47:32"
- [ ] Limited Spots: "7 de 25 Vagas Restantes"
- [ ] Expiring Bonus: "Bonus de R$2000 Apenas Hoje"

**CRITICAL VERIFICATION:**
- [ ] Scarcity is REAL
- [ ] Mechanism is truthful
- [ ] Not fake countdown that resets
- [ ] Not fake "spots remaining"

**Warning:** False scarcity = long-term trust destruction

## Quick Reference: Golden Rules

### The Meta-Rule
```
"A pagina nao vende. A pagina remove as razoes para NAO comprar."
```

### Show, Don't Tell
```
Para cada claim -> mostre prova
Para cada feature -> mostre beneficio
Para cada objecao -> mostre solucao
```

### Value Equation Attack
```
Section A: Capture attention (Dream Outcome + Urgency)
Section B: Build case (Likelihood + Value)
Section C: Remove friction (Minimize Effort)
```

### Optimization Priority
```
1. If scroll rate <40% -> Fix Section A
2. If conversion <3% -> Audit all 17 elements
3. If checkout abandon >25% -> Focus Section C
```

## Troubleshooting Quick Reference

| Problem | Check First | Quick Fix |
|---------|-------------|-----------|
| Low scroll | Headline | A/B test pain vs desire |
| No CTA clicks | Button text | Change to benefit + arrow |
| High bounce | Above fold | Add social proof |
| Low conversion | Value stack | Make gap more obvious |
| Cart abandon | Trust badges | Add security seals |
| Support overload | FAQ | Address real objections |
