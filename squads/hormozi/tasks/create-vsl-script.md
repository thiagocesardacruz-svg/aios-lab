## Purpose
Create a video sales letter script that converts cold traffic into buyers.

> "A VSL is a sales page that talks. Same structure, but with the power of voice and face." — Alex Hormozi

---

## VSL Structure (15-30 minutes)

### Part 1: Hook (0:00-2:00)
- Pattern interrupt
- Big promise
- Curiosity builder
- "Keep watching because..."

### Part 2: Story (2:00-6:00)
- Origin story (relatable struggle)
- Discovery moment
- First results
- Why sharing now

### Part 3: Content (6:00-15:00)
- Teach 3 key concepts
- Each with micro-story
- Build credibility through value
- Transition to offer naturally

### Part 4: Offer (15:00-20:00)
- What they get (stack)
- Value of each component
- Bonuses revealed
- Total value calculation

### Part 5: Close (20:00-25:00)
- Price reveal (anchored)
- Guarantee explanation
- Scarcity/urgency
- CTA (what to do next)

### Part 6: FAQ/Objections (25:00-30:00)
- Address top 5 objections
- Additional testimonials
- Final CTA

## Script Writing Formula

### Hook Template
```
What if I told you that [big claim]?

In the next [X] minutes, I'm going to show you exactly
how [avatar] are [achieving dream outcome] without
[common objection].

And if you stay until the end, I'll give you
[bonus/incentive to keep watching].
```

### Story Template
```
[X] years ago, I was [relatable struggle].

I tried [failed attempt 1], [failed attempt 2],
and [failed attempt 3]. Nothing worked.

Then I discovered [mechanism/insight].

Within [timeframe], I [achieved result].

Since then, I've helped [number] people do the same.

Today, I want to show you how...
```

### Offer Stack Template
```
Here's everything you get when you join today:

Component 1: [Name] — Value: $X
[Brief description of what it is and why it matters]

Component 2: [Name] — Value: $X
[Brief description]

[Continue for all components]

PLUS these bonuses:

Bonus 1: [Name] — Value: $X
Bonus 2: [Name] — Value: $X

Total Value: $[Sum]

But you won't pay anywhere near that...
```

## Execution Steps

### Step 1: Research Phase
- Watch competitor VSLs
- Identify winning hooks
- Note timing and pacing

### Step 2: Outline First
- Map all sections
- Time allocations
- Key points per section

### Step 3: Write Conversationally
- Read aloud while writing
- Short sentences
- Natural pauses marked

### Step 4: Add Visual Cues
```
[SLIDE: Headline text]
[B-ROLL: Customer testimonial]
[SCREEN RECORD: Demo]
```

### Step 5: Edit for Retention
- Remove fluff
- Add pattern interrupts
- Check hook every 2-3 minutes
