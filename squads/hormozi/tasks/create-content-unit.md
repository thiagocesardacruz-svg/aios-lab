## Purpose
Create a single content unit using the Hook-Retain-Reward framework.

> "The smallest amount of material necessary to hook, retain, and reward attention is a content unit." — Alex Hormozi

---

## The Content Unit Framework

```
┌─────────────────────────────────────────────────────────────────────────┐
│                       THE CONTENT UNIT                                  │
├──────────────────────┬──────────────────────┬───────────────────────────┤
│       HOOK           │       RETAIN         │       REWARD              │
├──────────────────────┼──────────────────────┼───────────────────────────┤
│ Make them notice     │ Make them consume    │ Satisfy their reason      │
│ Give reason to       │ Keep them curious    │ Exceed expectations       │
│ redirect attention   │ wanting more         │ Make them share           │
└──────────────────────┴──────────────────────┴───────────────────────────┘
```

## Part 1: HOOK

### Choose Topic from 5 Categories

| Category | Description | Example |
|----------|-------------|---------|
| Passado Distante | Life lessons from past | "What losing everything taught me" |
| Passado Recente | What you just did | "I tried X for 30 days" |
| Presente | Ideas as they come | "Thought while on a walk today" |
| Tendencias | Current events + your take | "What the news got wrong about X" |
| Fabricado | Created experiences | "I lived on $100 for a month" |

### Craft Headline with 7 Components

Use at least 2:
1. **Recencia** — Most recent possible
2. **Relevancia** — Personally significant to audience
3. **Celebridade** — Prominent people
4. **Proximidade** — Close to home
5. **Conflito** — Opposing ideas
6. **Incomum** — Strange, unique, rare
7. **Em andamento** — Still in progress

### Match Format to Platform

Study top content in your niche, match:
- Visual style
- Length
- Structure
- Editing style

---

## Part 2: RETAIN

### Use One or More Structures

**Lists**
- "7 Ways to..."
- "28 Things I Learned..."
- Flexible, curiosity-driven

**Steps**
- "How to X in 5 Steps"
- "The Exact Process I Use"
- Sequential, outcome-driven

**Stories**
- "What happened when..."
- "My journey from X to Y"
- Emotional, transformation-driven

### Interweave for Complex Content
- Lists within steps
- Stories to reinforce points
- Multiple ongoing storylines

---

## Part 3: REWARD

### Maximize Value Per Second
- No such thing as too long, only too boring
- Cut every sentence that doesn't add value
- Front-load the best stuff

### Fulfill the Promise
- If you promised "7 ways" deliver 7 good ways
- If you promised a result, deliver the path
- Never bait-and-switch

### Validate with Audience Growth
- If audience grows, content is working
- If stagnant, improve quality or increase volume
- "Se nao esta crescendo, suas coisas nao sao tao boas"

---

## Execution Steps

### Step 1: Choose Topic Category
Based on what you have available right now.

### Step 2: Write 5 Headlines
Using headline formula, pick the strongest.

### Step 3: Select Retention Structure
Lists, steps, stories, or combination.

### Step 4: Draft Content
Write conversationally, read aloud.

### Step 5: Edit for Value per Second
Cut ruthlessly. Every word must earn its place.

### Step 6: Format for Platform
Adapt to platform norms.

### Step 7: Post Manually
Do not pre-schedule. Manual posting performs better.
