## Direct Competitors

### 1. Mews

| Attribute | Details |
|-----------|---------|
| **Category** | Property Management System (PMS) |
| **Target** | Hotels, hostels, aparthotels |
| **Pricing** | €6-15/room/month |
| **Funding** | $185M Series C |
| **Strengths** | Modern PMS, good integrations, API-first |
| **Weaknesses** | Complex, expensive for small hotels, no AI |

**Positioning vs Us:**
- They're PMS, we're AI operations layer
- We integrate WITH systems like Mews
- We focus on tasks, they focus on data

## Market Gaps (Our Opportunity)

### 1. AI Operations Layer
- No one owns "AI for daily hotel operations"
- Everyone is PMS or channel manager
- We can be the "AI layer" that sits on top

### 2. Task-First Approach
- Competitors are system-first
- We're task-first (like Superhuman for email)
- "What do you need to do?" not "What data do you need?"

### 3. Affordable AI
- Enterprise AI is expensive
- ChatGPT is too generic
- We're specialized but accessible

### 4. Speed to Value
- Competitors need weeks to implement
- We can show value in hours
- "AI concierge in your inbox"
