## Copy Frameworks

### AIDA
- **A**ttention → **I**nterest → **D**esire → **A**ction

### PAS
- **P**roblem → **A**gitation → **S**olution

### BAB
- **B**efore → **A**fter → **B**ridge

### 4Ps
- **P**icture → **P**romise → **P**roof → **P**ush

## Funnel Types

| Type | Flow |
|------|------|
| Lead Magnet | Ad → Opt-in → Thank You → Email → Offer |
| Webinar | Ad → Register → Confirm → Reminders → Live → Offer |
| Sales | Ad → Sales Page → Checkout → Upsell → Thank You |
| Tripwire | Ad → Low-ticket → OTO 1 → OTO 2 → Core Offer |

## SEO Strategy

### Content Hierarchy
```
Pillar Content (2,500+ words)
    ↓
Cluster Content (1,500 words)
    ↓
Supporting Content (800 words)
```

### Target Metrics
| Metric | Target |
|--------|--------|
| Keyword in H1 | ✓ |
| Keyword in first 100 words | ✓ |
| Internal links | 3-5 |
| External links | 1-3 |
| Word count | 1,500-2,500 |
