# SEO On-Page Checklist

> Validation checklist for SEO-optimized content.

## Title & Meta

- [ ] Primary keyword in title tag
- [ ] Title under 60 characters
- [ ] Meta description includes keyword
- [ ] Meta description under 160 characters
- [ ] Meta description has CTA

## Headings

- [ ] Only one H1 on page
- [ ] H1 includes primary keyword
- [ ] H2s include secondary keywords
- [ ] Heading hierarchy is logical (H1 > H2 > H3)
- [ ] No skipped heading levels

## Content

- [ ] Keyword in first 100 words
- [ ] Keyword density 1-2% (not stuffed)
- [ ] LSI keywords included naturally
- [ ] Content length matches competitor average
- [ ] Content fully addresses search intent

## Links

- [ ] 2-5 internal links to relevant pages
- [ ] External links to authoritative sources
- [ ] No broken links
- [ ] Anchor text is descriptive

## Images

- [ ] All images have alt text
- [ ] Alt text includes keywords where relevant
- [ ] Images are compressed
- [ ] File names are descriptive

## URL

- [ ] URL is short and clean
- [ ] URL includes primary keyword
- [ ] No special characters or numbers

## Technical

- [ ] Page loads in <3 seconds
- [ ] Mobile responsive
- [ ] No duplicate content
- [ ] Schema markup (if applicable)

## User Experience

- [ ] Content is scannable
- [ ] Bullet points and lists used
- [ ] Table of contents (for long content)
- [ ] Related content suggested
