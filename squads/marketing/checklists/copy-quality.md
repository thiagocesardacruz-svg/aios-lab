# Copy Quality Checklist

> Quality gate for all marketing copy before publication.

## Clarity

- [ ] Headline grabs attention in <3 seconds
- [ ] Main message is clear without jargon
- [ ] One idea per paragraph
- [ ] CTA is specific and actionable
- [ ] No ambiguous pronouns or references

## Persuasion

- [ ] Problem/pain clearly identified
- [ ] Solution clearly presented
- [ ] Benefits emphasized over features
- [ ] Social proof included
- [ ] Objections addressed
- [ ] Urgency/scarcity (if applicable)

## Voice & Tone

- [ ] Consistent with brand voice
- [ ] Appropriate for target audience
- [ ] No corporate speak or buzzwords
- [ ] Conversational where appropriate

## Technical

- [ ] No spelling errors
- [ ] No grammar issues
- [ ] Links work correctly
- [ ] Formatting is clean
- [ ] Mobile-friendly (short paragraphs)

## Compliance

- [ ] No false or misleading claims
- [ ] Testimonials are real and documented
- [ ] Disclaimers included where needed
- [ ] GDPR/privacy compliant

## SEO (if applicable)

- [ ] Primary keyword in headline
- [ ] Keywords in subheadings
- [ ] Meta description optimized
- [ ] Alt text for images

---

## Result

- [ ] **PASS** - Ready for publication
- [ ] **REVISE** - Minor changes needed
- [ ] **REWRITE** - Major issues found
