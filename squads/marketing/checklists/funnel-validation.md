# Funnel Validation Checklist

> Quality gate for all funnels before launch.

## Pre-Launch Validation

### Strategy Alignment

- [ ] Funnel type matches business objective
- [ ] ICP clearly defined and documented
- [ ] Offer matches ICP pain points
- [ ] Price point validated
- [ ] Traffic source identified
- [ ] Budget allocated

### Page Flow

- [ ] All pages exist and are accessible
- [ ] Navigation flow is logical
- [ ] No dead ends in funnel
- [ ] Exit points are intentional
- [ ] Mobile responsive on all pages

---

## Landing Page

### Above the Fold

- [ ] Headline grabs attention (<3 sec)
- [ ] Value proposition is clear
- [ ] CTA is visible without scrolling
- [ ] Lead magnet/offer is compelling
- [ ] No distractions (clean design)

### Copy Quality

- [ ] Passes copy-quality checklist
- [ ] Benefits over features
- [ ] Social proof present
- [ ] Objections addressed
- [ ] CTA is specific and actionable

### Technical

- [ ] Form works correctly
- [ ] Validation messages clear
- [ ] Success redirect works
- [ ] Thank you page loads
- [ ] Double opt-in (if required)

---

## Thank You Page

- [ ] Confirms opt-in/purchase
- [ ] Delivers promised value
- [ ] Sets expectations
- [ ] Provides next steps
- [ ] Upsell/cross-sell (if applicable)

---

## Email Sequence

### Setup

- [ ] All emails created
- [ ] Sequence timing correct
- [ ] Triggers configured
- [ ] Segments defined
- [ ] Unsubscribe works

### Content

- [ ] Subject lines compelling
- [ ] Preview text optimized
- [ ] Personalization working
- [ ] Links correct
- [ ] CTAs clear

### Compliance

- [ ] Sender info correct
- [ ] Physical address included
- [ ] Unsubscribe easy
- [ ] Privacy policy linked

## Sales Page

### Structure

- [ ] Headline hooks immediately
- [ ] Problem clearly identified
- [ ] Solution presented compellingly
- [ ] Benefits stacked
- [ ] Social proof abundant
- [ ] Guarantee prominent
- [ ] FAQ addresses objections
- [ ] Multiple CTAs placed

### Copy

- [ ] Passes copy-quality checklist
- [ ] Reads naturally
- [ ] No jargon
- [ ] Urgency (ethical)
- [ ] Clear pricing

### Technical

- [ ] CTA buttons work
- [ ] Scroll to checkout works
- [ ] Mobile layout correct
- [ ] Fast load time (<3s)
