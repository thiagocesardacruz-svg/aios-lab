# Customer Squad

> Post-Sale Success, Retention & LTV

## Overview

The Customer Squad ensures customers achieve success with Travel Tech Digital products. Responsible for:

- Onboarding experience
- Customer retention
- Churn prevention
- Upsell and expansion
- Customer advocacy

## Agents

| Agent | Role | Focus |
|-------|------|-------|
| **Haven** (customer-lead) | Lead | Success & retention |
| **Guide** (onboarding-architect) | Onboarding | Time-to-value |
| **Renew** (retention-strategist) | Retention | Churn & upsell |

## Commands

| Command | Description |
|---------|-------------|
| `/customer/onboarding-sequence` | Create onboarding |
| `/customer/retention-system` | Design retention system |
| `/customer/upsell-campaign` | Create upsell campaign |

## Customer Lifecycle

```
Purchase → Onboarding → Adoption → Expansion → Advocacy
    ↓          ↓            ↓          ↓          ↓
 Welcome   First Value    Habits    Upsell    Referrals
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Time to First Value | < 3 min |
| Day 7 Retention | > 60% |
| NPS | > 50 |
| Churn Rate | < 5% |
| Expansion Revenue | > 20% |
