## Pre-Flight Checklist

### Before Creating Any Ad

- [ ] Read goated-ads-sop.md completely
- [ ] Understand Ad Assembly process (50 hooks x 3-5 meats x 1-3 CTAs)
- [ ] Friday blocked as dedicated advertising day
- [ ] 3-hour block reserved for production session
- [ ] Testing infrastructure ready (ad platform, pixel, tracking)

### Red Flags - STOP if:

- [ ] Budget <$50/day per ad set (insufficient data)
- [ ] No CAC calculation done
- [ ] No avatar research completed
- [ ] Haven't studied Eugene Schwartz awareness levels
- [ ] Expecting results from <100 conversions

## Pre-Requisites Checklist

### Fundamental Math (All Required)

- [ ] CAC maximum calculated: LTGP / 3 = $___
- [ ] Break-even in days defined (target: <30 days)
- [ ] Test budget calculated: 10x expected CPA = $___
- [ ] Minimum margin validated: ___% (must be >30% after ads)
- [ ] Daily minimum per ad set: $50+

### Market Research (All Required)

- [ ] 10 competitors analyzed in Meta Ad Library
- [ ] 5 working angles identified and documented
- [ ] 3 dominant offers in market mapped
- [ ] Market gaps documented
- [ ] Competitor ad frequency noted

### Surgical Avatar (All Required)

- [ ] Pain #1 identified (the one that keeps them awake)
- [ ] Exact language collected (screenshots of comments)
- [ ] Awareness level measured (1-5 scale)
- [ ] Purchasing power validated
- [ ] Demographics documented (age range minimum 10 years)

### Avatar Language Verification

- [ ] Minimum 10 real comments/quotes collected
- [ ] Language audit: 80%+ of copy uses avatar words
- [ ] Pain vocabulary documented
- [ ] Desire vocabulary documented
- [ ] Objection vocabulary documented

## Hook Creation Checklist (80% of Effort)

### Pre-Hook Research

- [ ] Review previous winning hooks from past campaigns
- [ ] Pull hooks from organic content (longs, shorts, tweets, emails)
- [ ] Save competitor ads to phone album
- [ ] Identify viral content hooks in niche
- [ ] Check Ad Libraries as last resort

### Awareness Level Distribution

- [ ] Level 5 - Most Aware (Offer): ___ hooks
- [ ] Level 4 - Product-Aware (Proof): ___ hooks
- [ ] Level 3 - Solution-Aware (Promise): ___ hooks
- [ ] Level 2 - Problem-Aware (Pain): ___ hooks
- [ ] Level 1 - Unaware (Curiosity): ___ hooks
- [ ] Total hooks = 50 (minimum)

### Distribution Rule Check

- [ ] Verify NOT 90%+ in "Most Aware" bucket
- [ ] Spread hooks across awareness levels
- [ ] When in doubt, went broader (capture warm + attract cold)

### Hook Quality Checklist (Each Hook)

- [ ] Specific problem mentioned
- [ ] Number in first line (when applicable)
- [ ] Pattern interrupt element present
- [ ] Avatar speaking to avatar
- [ ] Under 3 seconds when spoken

### Level 5 Hooks Created (Offer)

- [ ] Discount/deal hook created
- [ ] New feature/formula hook created
- [ ] Limited time hook created
- [ ] Best-seller hook created
- [ ] Upgrade/improvement hook created

### Level 4 Hooks Created (Proof)

- [ ] Social proof quantity hook created
- [ ] Industry/vertical proof hook created
- [ ] Testimonial teaser hook created
- [ ] Case study hook created
- [ ] Results/ROI hook created

### Level 3 Hooks Created (Promise)

- [ ] Speed/fastest way hook created
- [ ] Result doubling hook created
- [ ] Introducing product hook created
- [ ] Method reveal hook created
- [ ] Outcome promise hook created

### Level 2 Hooks Created (Pain)

- [ ] "Tired of X" hook created
- [ ] Missing elements hook created
- [ ] Better way hook created
- [ ] Problem cost hook created
- [ ] Pain amplification hook created

### Level 1 Hooks Created (Curiosity)

- [ ] Hidden danger hook created
- [ ] Unexpected loss hook created
- [ ] Secret reveal hook created
- [ ] Myth-busting hook created
- [ ] Pattern interrupt curiosity hook created

### Meme/Culture Hooks (Bonus)

- [ ] Niche-specific meme hooks considered
- [ ] Culture-specific references used where appropriate
- [ ] Meme-like content format tested

### CTA 1 - Complete Anatomy

- [ ] What to do: action specified
- [ ] How to do it: instruction clear
- [ ] When to do it: urgency element
- [ ] What they get: benefit stated
- [ ] What happens next: next step explained

### CTA 2 - Complete Anatomy

- [ ] What to do: action specified
- [ ] How to do it: instruction clear
- [ ] When to do it: urgency element
- [ ] What they get: benefit stated
- [ ] What happens next: next step explained

- [ ] What to do: action specified
- [ ] How to do it: instruction clear
- [ ] When to do it: urgency element
- [ ] What they get: benefit stated
- [ ] What happens next: next step explained

- [ ] Clear > Clever principle applied

## Campaign Configuration Checklist

### Campaign Structure

- [ ] CBO (Campaign Budget Optimization) enabled
- [ ] Maximum 3-5 ad sets configured
- [ ] Budget minimum $50/day per ad set
- [ ] Pixel installed correctly
- [ ] All conversion events configured

### Targeting Settings

- [ ] Broad targeting (3-10% of population)
- [ ] NO stacked interests (let algorithm work)
- [ ] Age range: minimum 10-year spread
- [ ] Lookalike 1% from customer list created
- [ ] Exclusions properly set

### Copy Elements

- [ ] Headline: specific result + timeframe
- [ ] Primary text: <125 characters
- [ ] Description: main benefit
- [ ] CTA button: action verb (Garantir, Comecar, Acessar)

### Landing Page Alignment

- [ ] Headline matches ad exactly
- [ ] CTA visible in <3 seconds of scroll
- [ ] Social proof visible above fold
- [ ] Load time <2 seconds
- [ ] Conversion rate baseline: minimum 2%

## Day 1 Monitoring Checklist (First 24 Hours)

### KPI Checks

- [ ] CTR measured: ___% (threshold: >1%)
- [ ] CPM measured: $___ (threshold: <$30)
- [ ] CPC measured: $___ (threshold: <$3)
- [ ] Link CTR measured: ___% (threshold: >0.8%)

### Diagnostic Actions

- [ ] If CTR <1%: Hook problem identified
- [ ] If CPM >$30: Creative quality issue flagged
- [ ] If CPC >$3: Relevance problem noted
- [ ] If Link CTR <0.8%: Copy weakness marked

## Red Flags - Immediate Action Checklist

### Kill Signals

- [ ] CPA doubled: PAUSA imediata
- [ ] CTR <0.5%: MATA o ad
- [ ] Frequency >10: REFRESH criativo
- [ ] ROAS <2x for 3 days: ANALISE profunda
- [ ] Relevance score <4: NOVO CRIATIVO

### Troubleshooting: CPM Alto

- [ ] Creative fatigue checked (frequency >7?)
- [ ] Relevance score checked (>6?)
- [ ] Competition spike checked (Q4, Black Friday?)
- [ ] Account penalty checked

### Troubleshooting: CPA Subindo

- [ ] Landing page conversion checked
- [ ] Offer strength evaluated
- [ ] Competitor offers analyzed
- [ ] Algorithm changes considered

### Troubleshooting: CTR Baixo

- [ ] Hook quality reviewed
- [ ] Visual uniqueness assessed
- [ ] Copy resonance tested
- [ ] Market saturation considered

## Scale Signals Checklist

### Ready to Scale (All Must Be True)

- [ ] ROAS >3x for 7 consecutive days
- [ ] CPA stable for 500+ conversions
- [ ] Frequency <3 after 14 days
- [ ] CTR maintains >1%

### Scale Methods

- [ ] 20% Rule applied: budget +20% every 48 hours
- [ ] New ad sets: same copy, new audience
- [ ] Surfing: pause Friday, reactivate Monday
- [ ] Day-parting: off overnight if B2B
