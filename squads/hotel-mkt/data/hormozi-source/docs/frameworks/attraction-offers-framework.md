## OFERTA #1: GANHE SEU DINHEIRO DE VOLTA

### _"Se você fizer X ou alcançar Y em Z dias, você pode ter de graça"_

### BLUEPRINT DE IMPLEMENTAÇÃO

**Fórmula Core:**

```
INVESTIMENTO INICIAL (100% reembolsável)
+ CRITÉRIOS CLAROS DE QUALIFICAÇÃO
+ PRAZO DEFINIDO
= ZERO RISCO PERCEBIDO
= 2-3X MAIS CONVERSÕES
```

### SISTEMA DE 5 PASSOS

**PASSO 1: DEFINA O INVESTIMENTO**

- Valor: 3-5x seu custo de entrega
- Sweet spot: $300-$3,000 (B2C) | $3,000-$30,000 (B2B)
- Regra: Alto o suficiente para filtrar, baixo o suficiente para acessível

**PASSO 2: ESTABELEÇA CRITÉRIOS (Escolha um modelo)**

**Modelo A - BASEADO EM RESULTADOS:**

```
□ Perder X quilos em Y semanas
□ Conseguir Z clientes em 30 dias
□ Aumentar receita em X% em 90 dias
□ Atingir métrica específica mensurável
```

_Use quando: Cliente tem habilidade mas precisa motivação_

**Modelo B - BASEADO EM AÇÕES:**

```
□ Comparecer a 100% das sessões
□ Completar 100% das tarefas
□ Postar progresso semanalmente  
□ Fazer check-ins diários no app
□ Participar de todas as reuniões
```

_Use quando: Cliente precisa desenvolver hábitos_

**Modelo C - HÍBRIDO (MAIS EFICAZ):**

```
□ Ações obrigatórias (70% do peso)
□ Resultado mínimo (30% do peso)
□ Divulgação social incluída
```

_Use quando: Quer garantir sucesso E marketing_

**PASSO 3: ESTRUTURE O PRAZO**

- Curto (21-28 dias): Urgência máxima, momentum alto
- Médio (6-8 semanas): Equilíbrio ideal
- Longo (12-16 semanas): Transformações profundas

**PASSO 4: APLIQUE O CRÉDITO (3 Opções)**

**Opção A - Crédito Total Imediato:**

- 100% aplicado na próxima compra
- Melhor para: Tickets altos, clientes premium

**Opção B - Crédito Distribuído:**

```
$600 ganhos = $50/mês por 12 meses
Benefício: Cliente paga mas com desconto perpétuo
Resultado: Maior LTV, menor churn
```

**Opção C - Escolha do Cliente:**

- Crédito total AGORA para programa premium
- OU distribuído para continuidade
- Benefício: Mais controle percebido = mais satisfação

**PASSO 5: FAÇA TODOS VENCEDORES (Script Mágico)**

_No meio do programa:_

> "Sei que você está tentando atingir [objetivo de curto prazo], mas qual é seu objetivo de longo prazo?
> 
> Ok, isso é ótimo ouvir. Você entende que não é sobre este programa de curto prazo, mas sobre seus resultados de longo prazo.
> 
> Então vou te dizer o quê. Para mostrar o quanto quero que você atinja seu objetivo de longo prazo, vou creditar este programa para o próximo, QUER VOCÊ ATINJA O OBJETIVO OU NÃO.
> 
> Como isso soa?"

## OFERTA #2: SORTEIOS

### _"Muitos entrarão, muitos ganharão"_

### ARQUITETURA DO SORTEIO PERFEITO

**Psicologia Core:** Todos acham que vão ganhar + FOMO de não participar + Consolação melhor que nada = 40-70% conversão dos "perdedores"

### SISTEMA DE 6 PASSOS MATADOR

**PASSO 1: ESCOLHA O GRANDE PRÊMIO**

```
REGRA DE OURO: 
Grande Prêmio = Exatamente o que você quer vender depois
Valor Percebido = 10-20x o preço do ticket médio

Exemplos Calibrados:
- Curso de $10k → Bolsa integral
- Tratamento de $6k → Procedimento completo grátis  
- Consultoria de $25k → 6 meses grátis
```

**PASSO 2: CRIE A OFERTA PROMOCIONAL (O Segredo)**

```
"Bolsa Parcial" = 30-50% desconto do valor total
Mas apresente como 70-90% de desconto

Matemática:
Prêmio de $10,000
"Bolsa parcial" de $7,000 (parece 70% desconto)
Cliente paga $3,000 (sua meta de venda real)
```

**PASSO 3: ESTRUTURE A CAPTURA**

```
INFORMAÇÕES OBRIGATÓRIAS:
□ Nome completo
□ Email
□ Telefone celular
□ Pergunta qualificadora

PERGUNTA MÁGICA:
"Por que você merece ganhar?"
(Revela dor, urgência e comprometimento)
```

**PASSO 4: DEFINA O CRONOGRAMA**

```
Dia 1-3: Divulgação massiva
Dia 4-5: "Últimas vagas" 
Dia 6: "Último dia!"
Dia 7: Anúncio do vencedor
Dia 7-14: Conversão dos "perdedores"
```

**PASSO 5: EXECUTE O ANÚNCIO**

```
PÚBLICO:
- Post celebrando vencedor
- Tag em stories
- Email para toda base

PRIVADO (onde o dinheiro está):
- Email/SMS para "perdedores"
- Ligação em 24h
- Oferta expira em 48-72h
```

**PASSO 6: CONVERTA OS "PERDEDORES"**

**Script de Conversão Master:**

> "Oi [Nome], primeiro PARABÉNS!
> 
> Sei que você não ganhou a bolsa integral, mas suas respostas foram TÃO BOAS que consegui uma bolsa parcial de $[X] para você!
> 
> Normalmente é $[preço cheio], mas com a bolsa fica apenas $[seu preço real].
> 
> Mas preciso da resposta até [amanhã] porque só consegui [3] bolsas parciais.
> 
> Você consegue aproveitar?"

### HACKS AVANÇADOS DE SORTEIOS

**HACK #1: DUPLO VENCEDOR** "Se alguém que você indicar ganhar, você também ganha!" Resultado: 3-5x mais participantes

**HACK #2: QUALIFICAÇÃO PRÉVIA** "Apenas para [público específico]" Resultado: Leads 10x mais qualificados

**Princípio:** Quanto maior a diferença entre isca e premium, mais pessoas compram premium

□ Suporte personalizado
□ Garantias
□ Bônus premium
□ Acesso prioritário
□ Customização
□ Velocidade/Conveniência

MANTENHA APENAS:
✓ Função core básica
✓ Resultado mínimo viável
```

**PASSO 2: CRIE A PREMIUM (Loaded)**

```
ADICIONE TUDO:
✓ 1-on-1 ilimitado
✓ Garantia de resultados
✓ Todos os bônus
✓ Acesso vitalício
✓ Personalização total
✓ Entrega expressa
✓ Suporte prioritário 24/7
```

**PASSO 3: PRECIFIQUE ESTRATEGICAMENTE**

```
ISCA: Custo real + 10% margem
PREMIUM: 3-5x preço da isca

Exemplo Real:
Isca: $497 (margem mínima)

## OFERTA #4: COMPRE X, GANHE Y GRÁTIS

### _"Matemática criativa para o cérebro do comprador"_

### A CIÊNCIA DO "GRÁTIS"

**Fórmula Universal:**

```
PREÇO TOTAL ÷ ITENS TOTAIS = PREÇO POR ITEM
MAS APRESENTE COMO:
PREÇO TOTAL = PREÇO DE X ITENS (Y itens "grátis")
```

### MATRIZ DE CONFIGURAÇÕES

**CONFIGURAÇÃO A: MESMO PRODUTO**

```
Ruim: 3 por $300 ($100 cada)
Bom: Compre 1 por $300, ganhe 2 grátis
Melhor: Compre 1, ganhe 5 grátis por $300
```

**CONFIGURAÇÃO B: PRODUTOS DIFERENTES**

```
Compre X (principal) + Ganhe Y, Z (complementares)

Exemplo:
Compre tratamento facial ($500)
Ganhe limpeza + hidratação grátis
(Total real: $500, valor percebido: $800)
```

**CONFIGURAÇÃO C: TEMPO/SERVIÇO**

```
Compre 3 meses, ganhe 3 meses grátis
Compre 10 sessões, ganhe 10 sessões grátis
Compre hoje, ganhe 1 ano de suporte grátis
```

### TABELA DE CONFIGURAÇÃO ÓTIMA

|Margem Bruta|Config Recomendada|Exemplo|
|---|---|---|
|<30%|Compre 2, Ganhe 1|Produtos físicos|
|30-60%|Compre 1, Ganhe 1|Serviços locais|
|60-80%|Compre 1, Ganhe 2|Digital/Cursos|
|>80%|Compre 1, Ganhe 3+|Software/Info|

### CALENDÁRIO DE IMPLEMENTAÇÃO

**PREPARAÇÃO (Dia -7):**

- Calcule margem real por unidade

**LANÇAMENTO (Dia 1):**

- Email: "Oferta especial limitada"
- Destaque: quantidade grátis > paga

**URGÊNCIA (Dia 3-5):**

- "Restam apenas X kits"
- Contador regressivo visual

**FECHAMENTO (Dia 7):**

- "Última chance - acaba hoje!"
- Bônus extra para decisão imediata
