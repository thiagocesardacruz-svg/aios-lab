## Technical Localization

### Date & Time
- [ ] Date format correct (DD/MM/YYYY vs MM/DD/YYYY)
- [ ] Time format correct (12h vs 24h)
- [ ] Time zone handling correct
- [ ] Calendar system appropriate (Gregorian/Hijri)

### Numbers & Currency
- [ ] Number format correct (1,000.00 vs 1.000,00)
- [ ] Currency symbol correct
- [ ] Currency placement correct
- [ ] Decimal separator correct
- [ ] Thousand separator correct

### Measurements
- [ ] Units appropriate (metric vs imperial)
- [ ] Temperature scale correct (C vs F)
- [ ] Distance units correct
- [ ] Weight units correct
- [ ] Volume units correct

### Contact Information
- [ ] Phone format correct
- [ ] Address format correct
- [ ] Postal code format correct
- [ ] Country code included
