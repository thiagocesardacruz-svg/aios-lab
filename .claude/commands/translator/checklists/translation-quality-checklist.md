# Translation Quality Checklist

## Identity
- **ID:** translation-quality-checklist
- **Squad:** translator
- **Type:** checklist
- **Version:** 1.0.0

---

## Pre-Translation

### Source Analysis
- [ ] Source content fully understood
- [ ] Content type identified (marketing/technical/legal/UI)
- [ ] Target audience identified
- [ ] Tone and style requirements clear
- [ ] Character/word limits noted (if any)

### Preparation
- [ ] Glossary loaded and reviewed
- [ ] Style guide consulted
- [ ] Reference materials available
- [ ] Context/screenshots available (for UI)
- [ ] Questions to client resolved

---

## During Translation

### Accuracy
- [ ] All content translated (nothing omitted)
- [ ] Meaning accurately conveyed
- [ ] No unintended additions
- [ ] Numbers and data verified
- [ ] Names and proper nouns handled correctly
- [ ] Dates and formats localized

### Terminology
- [ ] Glossary terms used consistently
- [ ] Technical terms accurate
- [ ] Brand terms preserved/adapted correctly
- [ ] Industry-specific terms verified
- [ ] New terms documented for glossary

### Language Quality
- [ ] Grammar correct
- [ ] Spelling verified
- [ ] Punctuation appropriate for target language
- [ ] Sentence structure natural
- [ ] No awkward phrasing

### Tone & Style
- [ ] Tone matches original intent
- [ ] Formality level appropriate
- [ ] Style consistent throughout
- [ ] Brand voice maintained
- [ ] Cultural register correct

---

## Post-Translation

### Self-Review
- [ ] Full read-through completed
- [ ] Compared against original
- [ ] Checked for typos
- [ ] Verified formatting
- [ ] Placeholder integrity confirmed

### Technical Check
- [ ] Placeholders intact ({name}, %s, etc.)
- [ ] HTML/markup preserved correctly
- [ ] Links functional
- [ ] Character limits respected
- [ ] File format correct

### Cultural Review
- [ ] No offensive content
- [ ] Idioms properly adapted
- [ ] Cultural references appropriate
- [ ] Local conventions followed
- [ ] Sensitive topics handled appropriately

---

## Final Validation

### Completeness
- [ ] All sections translated
- [ ] No "[TO BE TRANSLATED]" markers left
- [ ] Metadata updated
- [ ] Translation notes documented
- [ ] Glossary updates listed
