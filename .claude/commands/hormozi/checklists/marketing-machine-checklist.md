## Pre-Flight Checklist

Before starting Marketing Machine operations, verify:

### Infrastructure
- [ ] Call recording system active
- [ ] Extra storage purchased (no auto-delete)
- [ ] Social monitoring tools configured
- [ ] Content library/folder structure created
- [ ] Video editing capability available
- [ ] Community platform active
- [ ] Review site accounts verified

### Legal & Compliance
- [ ] Permission language in all customer contracts
- [ ] "Compliance, training, and advertising purposes" covered
- [ ] Process to ask customers if OK with using story
- [ ] Protocol to honor "no" responses

### Team Readiness
- [ ] Sales trained to identify happy customers weekly
- [ ] Support reps know to screenshot wins
- [ ] Community manager knows wins collection process
- [ ] Camera teams identified for events
- [ ] Video editing resources available

### Content Ready
- [ ] Hashtags and tags defined for customers
- [ ] 6-point testimonial script documented
- [ ] 10 man-in-street questions prepared
- [ ] Bonus offers designed (big and sexy)
- [ ] Award trophy designed (craved, not boring)

## Node III: In-Person Events Checklist

### Pre-Event Setup
- [ ] Recording room reserved (separate from event)
- [ ] OR 2-3 hotel rooms for recording
- [ ] Camera teams hired:
  - [ ] Team 1: Event recording
  - [ ] Team 2: Testimonials (better team)
  - [ ] Team 3: Man-in-street (if doing both)
- [ ] Runner hired to bring clients

### Equipment
- [ ] Lighting set up and tested
- [ ] Sound tested
- [ ] Multiple mics available
- [ ] Multiple cameras available
- [ ] Backup equipment ready
- [ ] **TEST. TEST. TEST.** everything

### Hit List Preparation
- [ ] Priority customers identified
- [ ] Confirmed in writing
- [ ] Hit list printed on clipboard
- [ ] Check-off column ready
- [ ] Handler incentive defined ($50/extra capture)

### Event Announcement
- [ ] Testimonial process announced at start
- [ ] Handler introduced to all attendees
- [ ] Pull timing communicated (not during keynotes)

### Stage Ads Capture
Timing: End of day 1 or before lunch day 2
- [ ] Picture on stage with crowd celebrating behind
- [ ] Multiple selfie images
- [ ] Video (one take - make it count!)

### Professional Testimonials
- [ ] Customers not pulled during keynotes
- [ ] Alternative times: morning, lunch, breaks, end of day
- [ ] Customer bonus for participation offered
- [ ] 6-point script followed:

**The 6-Point Script:**
```
1. (Internal Struggle) What was rock bottom before doing this?
   What did you have to do that you hated?
   What things did you want to do that you couldn't?

2. (External Struggle) What were your objective metrics?

3. (Skepticism) What were your main concerns about buying?

4. (Overcome) What made you do it anyways?

5. (External Victory) What are your objective metrics now?

6. (Internal Victory) What's been the best moment since?
   What can you do now you couldn't do before?
   What things did you hate doing you don't have to do anymore?
```

- [ ] Hook: Multiple versions tested before main story
- [ ] Name stated at beginning of video
- [ ] Videos saved, uploaded, music added

### Man-in-Street Testimonials
- [ ] 10 core questions prepared
- [ ] Target: 10+ responses per question
- [ ] Questions printed for camera team
- [ ] Tally space below each question
- [ ] Ask only one question until target reached

**Example Questions:**
```
- How many new customers have you gained since you started?
- What were you making before X vs now?
- How much extra income has X added?
- How much weight have you lost since you started here?
- How much has your credit score improved?
- What was your old score and what is it now?
```

- [ ] Fast-hitting highlight reels created

## Node V: Bonuses & Awards Checklist

### Bonus Unlock Creation
- [ ] Valuable deliverable created:
  - [ ] Training content
  - [ ] Project/deliverable
  - [ ] One-time access
  - [ ] Status tier
- [ ] "Big and sexy" requirement met
- [ ] Price attached (for sale option)
- [ ] Testimonial unlock option created

### Bonus Unlock Promotion
- [ ] Pitch video recorded
- [ ] Placed where all customers see it
- [ ] Unlock instructions provided:
  - [ ] Video under 5 min hitting 6 points
  - [ ] Email to proof@yourco.com
  - [ ] Reference video (best one) shared
  - [ ] Post in group with stats required
  - [ ] Tag community manager required
  - [ ] Keywords included for scraping

### Award System Creation
- [ ] Aspirational goal identified
- [ ] Award designed:
  - [ ] Tied to community identity
  - [ ] Big and cool (money spent)
  - [ ] People crave it
  - [ ] NOT boring
- [ ] Video announcing award created
- [ ] Milestone requirements defined
- [ ] Public recognition at event planned

### Four-Form Proof Capture
For each award recipient:
- [ ] **PROOF 1:** Post showing actual numbers in group
  - [ ] Template used
  - [ ] Community owner tagged
- [ ] **PROOF 2:** Video sent to proof@yourco.com
- [ ] **PROOF 3:** Unboxing video when award received
  - [ ] Personalized/engraved award mailed
- [ ] **PROOF 4:** Stage/in-person presentation recorded
  - [ ] Generic award for stage presentation ready

  - [ ] Video attachment required

- [ ] Announced via ALL internal channels:
  - [ ] Email
  - [ ] Group messages
  - [ ] Community posts
- [ ] Countdown to deadline started
- [ ] Personal invitations to top 50 customers

- [ ] Conversions counted at end

## Troubleshooting Quick Reference

### No Testimonials Coming In
- [ ] Check award crave-ability (big and cool?)
- [ ] Simplify unlock process
- [ ] Add rewards for team screenshot submission

### Founder Still in Most Ads
- [ ] Activate more nodes
- [ ] Test more customer content
- [ ] Set 80% customer ratio mandate

### Events Not Producing Content
- [ ] Review equipment TEST records
- [ ] Count planned vs captured testimonials
- [ ] Review timing (end of day 1, before lunch day 2)

### Competition Fatigue
- [ ] Confirm frequency <2x/year
- [ ] Improve award attractiveness
- [ ] Reach out to new customers

### Team Not Collecting Screenshots
- [ ] Add gift card incentive
- [ ] Manager kudos required
- [ ] "If you don't reward it, they won't do it"
