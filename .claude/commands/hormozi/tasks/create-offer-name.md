## Purpose
Create a compelling offer name using the MAGIC naming framework.

> "A great name does half the selling for you. A bad name makes everything harder." — Alex Hormozi

---

## The MAGIC Framework

### M — Make a Magnetic Reason Why
The name should create curiosity or give a reason to engage.
- "Launch Week Special" (reason = timing)
- "Founders Edition" (reason = exclusivity)
- "Beta Access" (reason = early adoption)

### A — Announce Your Avatar
The name should call out exactly who it's for.
- "Gym Owners..."
- "Coaches Who Want..."
- "The Agency Blueprint"

### G — Give Them a Goal
The name should promise a destination.
- "...to Six Figures"
- "...Sales Machine"
- "...Client Magnet"

### I — Indicate a Time Interval
The name should set expectation for speed.
- "30-Day..."
- "...in 90 Days"
- "The 6-Week..."

### C — Complete with Container Word
The name should categorize what it is.
- System, Blueprint, Framework
- Academy, Program, Masterclass
- Challenge, Bootcamp, Sprint

---

## Naming Formula

```
[Magnetic Reason] + [Avatar] + [Goal] + [Time] + [Container]
```

### Examples

| Components | Name |
|------------|------|
| Avatar + Goal + Container | "Gym Owner's Client Acquisition System" |
| Time + Avatar + Goal | "6-Week Agency Scaling Blueprint" |
| Reason + Goal + Container | "Founders Edition Sales Machine" |
| Avatar + Time + Goal | "Coaches' 30-Day Lead Generation Sprint" |

---

## Execution Steps

### Step 1: Define Components

```yaml
components:
  avatar: ""           # Who is this for?
  goal: ""             # What do they achieve?
  time_interval: ""    # How fast?
  reason_why: ""       # Why now/this?
  container_word: ""   # What type of thing is it?
```

### Step 2: Generate 10 Combinations
Mix and match components in different orders.

### Step 3: Test Against Criteria

| Criteria | Check |
|----------|-------|
| Says who it's for | ☐ |
| Says what they get | ☐ |
| Creates curiosity | ☐ |
| Easy to say/remember | ☐ |
| Not easily confused | ☐ |
| Available as domain | ☐ |

### Step 4: Get Feedback
Test top 3 names with target audience.

### Step 5: Final Selection
Choose based on feedback and gut.

## Common Naming Mistakes

1. **Too generic** — "Business Course" (says nothing)
2. **Too clever** — Inside jokes no one gets
3. **Too long** — More than 5-6 words
4. **Doesn't match audience** — Corporate name for beginners
5. **Trademark conflicts** — Someone else owns it
