# Analysis Phase Checklist

## Phase Overview
**Objective:** Analyze market, competition, and identify gaps
**Duration:** Quick (2h) / Standard (1 day) / Deep (2-3 days)
**Prerequisites:** Discovery phase complete

---

## Market Analysis

### Market Definition
- [ ] Define broad market
- [ ] Define target market
- [ ] Define niche
- [ ] Set geographic focus
- [ ] Set time horizon

### Market Sizing
- [ ] Calculate TAM (Total Addressable Market)
- [ ] Calculate SAM (Serviceable Addressable Market)
- [ ] Calculate SOM (Serviceable Obtainable Market)
- [ ] Document methodology and sources

### Market Trends
- [ ] Identify 3-5 macro trends
- [ ] Assess impact of each trend
- [ ] Identify technology trends
- [ ] Identify economic trends
- [ ] Identify social/behavioral trends
- [ ] Identify regulatory trends

### Market Maturity
- [ ] Assess Schwartz sophistication level (1-5)
- [ ] Document evidence for level
- [ ] Define communication implications

### Entry Analysis
- [ ] List entry barriers
- [ ] List entry enablers
- [ ] Assess overall difficulty

**Output:** `market-analysis.md`

---

## Competitor Mapping

### Identification
- [ ] List direct competitors (3-5)
- [ ] List indirect competitors (3-5)
- [ ] List alternative solutions (DIY, do nothing, etc.)

### Deep Dive (Top 3-5 competitors)
- [ ] Document overview (founding, size, funding)
- [ ] Analyze product offering
- [ ] Analyze pricing
- [ ] Analyze key features
- [ ] Identify strengths
- [ ] Identify weaknesses
- [ ] Identify opportunities against them

### Comparison
- [ ] Create feature comparison matrix
- [ ] Create positioning map
- [ ] Analyze messaging patterns

### Messaging Analysis
- [ ] Document headlines
- [ ] Document value propositions
- [ ] Document proof points
- [ ] Identify common themes
- [ ] Identify messaging gaps

**Output:** `competitor-map.md`

---

## Gap Analysis

### Consolidation
- [ ] Review audience unmet needs
- [ ] Review market trends not addressed
- [ ] Review competitor gaps

### Gap Identification
- [ ] Identify functional gaps
- [ ] Identify emotional gaps
- [ ] Identify segment gaps
- [ ] Identify price gaps
- [ ] Identify access gaps
- [ ] Identify experience gaps
- [ ] Identify communication gaps

### SWOT
- [ ] List strengths (5+)
- [ ] List weaknesses (5+)
- [ ] List opportunities (5+)
- [ ] List threats (5+)
- [ ] Define SO strategies
- [ ] Define WO strategies
- [ ] Define ST strategies
- [ ] Define WT strategies

### Opportunity Scoring
- [ ] Score opportunities by impact
- [ ] Score opportunities by feasibility
- [ ] Score opportunities by urgency
- [ ] Prioritize opportunities

**Output:** `gap-opportunities.md`, `swot.md`

---

## Phase Complete Criteria

- [ ] Market is sized and understood
- [ ] Competition is mapped
- [ ] Gaps are identified and prioritized
- [ ] SWOT is complete
- [ ] Clear opportunities are defined
- [ ] Ready to move to Strategy phase
