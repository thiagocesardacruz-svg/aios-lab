# Strategy Phase Checklist

## Phase Overview
**Objective:** Define positioning and create strategic brief
**Duration:** Quick (2h) / Standard (4h) / Deep (1-2 days)
**Prerequisites:** Discovery and Analysis phases complete

---

## Positioning

### Foundation
- [ ] Define category (existing or new)
- [ ] Confirm primary target segment
- [ ] Confirm secondary target segment
- [ ] Identify unique differentiation

### Positioning Statement
- [ ] Write positioning statement using formula
- [ ] Test for clarity
- [ ] Test for uniqueness
- [ ] Test for believability
- [ ] Create 2-3 variations

### Value Proposition
- [ ] Map jobs to be done
- [ ] Map pains to relievers
- [ ] Map gains to creators
- [ ] Write value proposition statement
- [ ] Validate fit between customer and value

### Messaging Architecture
- [ ] Define brand promise
- [ ] Create 3-5 key messages
- [ ] Document proof points for each
- [ ] Create tagline options
- [ ] Align messaging to Schwartz level

### Competitive Positioning
- [ ] Confirm Schwartz level strategy
- [ ] Define how to communicate at that level
- [ ] Create differentiation statements

**Output:** `positioning-statement.md`

---

## Strategic Brief

### Document Assembly
- [ ] Write executive summary
- [ ] Compile problem & opportunity section
- [ ] Compile target audience section
- [ ] Compile competitive landscape section
- [ ] Compile positioning section

### Strategic Priorities
- [ ] Define priority 1 with metrics
- [ ] Define priority 2 with metrics
- [ ] Define priority 3 with metrics

### Success Metrics
- [ ] Define primary metrics
- [ ] Set targets for each
- [ ] Define timeframes

### Risks & Mitigations
- [ ] Identify top 3-5 risks
- [ ] Assess probability and impact
- [ ] Define mitigation strategies

### Go-to-Market Overview
- [ ] Define launch strategy
- [ ] Identify key channels
- [ ] Define initial pricing

### Next Steps
- [ ] Define immediate actions (2 weeks)
- [ ] Define short-term actions (30 days)
- [ ] Define medium-term actions (90 days)

**Output:** `strategic-brief.md`

---

## Validation

### Internal Review
- [ ] Review for completeness
- [ ] Review for consistency
- [ ] Review for clarity
- [ ] Review for actionability

### Stakeholder Review
- [ ] Present to stakeholders
- [ ] Gather feedback
- [ ] Incorporate changes
- [ ] Get sign-off

### Quality Check
- [ ] Problem is validated
- [ ] Audience is specific and researched
- [ ] Market opportunity is clear
- [ ] Competition is understood
- [ ] Positioning is unique
- [ ] Next steps are actionable

---

## Phase Complete Criteria

- [ ] Positioning statement is finalized
- [ ] Strategic brief is complete
- [ ] Stakeholders have approved
- [ ] Next steps are clear
- [ ] Ready to create PRD
