# Task: Problem Definition

## Objective
Transformar uma ideia ou oportunidade em um problema bem definido e validável.

## Context
Antes de pesquisar mercado ou concorrência, precisamos ter clareza absoluta sobre:
- Qual problema estamos resolvendo?
- Para quem?
- Por que agora?

## Process

### Step 1: Initial Capture
Responda sem filtro:
- O que você quer criar/resolver?
- O que te motivou a pensar nisso?
- Qual resultado ideal você imagina?

### Step 2: Problem Statement Canvas

```markdown
## Problem Statement

**Who:** [Quem tem esse problema?]

**What:** [Qual é o problema específico?]

**When:** [Quando/em que contexto ocorre?]

**Why it matters:** [Por que é importante resolver?]

**Current alternatives:** [Como resolvem hoje?]

**Why now:** [Por que é o momento certo?]
```

### Step 3: Job to Be Done
Preencha a frase:
> "Quando [situação], eu quero [motivação], para que eu possa [resultado esperado]."

### Step 4: Problem Validation Questions
- [ ] O problema é real ou imaginado?
- [ ] As pessoas pagam para resolver isso hoje?
- [ ] O problema é frequente ou eventual?
- [ ] A dor é forte o suficiente para mudar comportamento?
- [ ] Existe um segmento específico que sofre mais?

### Step 5: Scope Definition
- **In scope:** O que vamos resolver
- **Out of scope:** O que NÃO vamos resolver (por agora)
- **Assumptions:** O que estamos assumindo como verdade

## Output
- `problem-statement.md` preenchido
- Clareza sobre o problema central
- Lista de hipóteses para validar

## Prompts de Apoio

### Para Refinar o Problema
```
Atue como um product strategist experiente. Analise esta ideia inicial:
[IDEIA]

Ajude-me a:
1. Identificar o problema real por trás da ideia
2. Separar sintomas de causas raiz
3. Reformular como um Job to Be Done claro
4. Identificar 3 hipóteses críticas para validar
```

### Para Validar Relevância
```
Analise este problema:
[PROBLEMA]

Avalie criticamente:
1. É um problema real ou percebido?
2. Qual a frequência e intensidade da dor?
3. Quais soluções existem hoje? Por que são insuficientes?
4. Por que seria o momento certo para resolver isso?
```
