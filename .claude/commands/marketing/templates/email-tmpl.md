## Variations by Type

### Welcome Email
- Deliver promised value immediately
- Set expectations for future emails
- Provide quick win
- Soft CTA only

### Nurture Email
- Lead with value/education
- Tell stories
- Build relationship
- Position expertise

### Sales Email
- Clear offer
- Benefits focused
- Handle objections
- Strong CTA with urgency

### Cart Abandonment
- Show abandoned items
- Address concerns
- Add incentive
- Easy return to cart

### Re-engagement
- Acknowledge absence
- Remind of value
- Offer incentive
- Option to unsubscribe

- [ ] Subject line <50 characters

- [ ] One primary CTA
- [ ] Mobile-friendly (short paragraphs)
- [ ] Links work correctly
- [ ] Unsubscribe link present
- [ ] Personalization tags correct
- [ ] No spam trigger words
- [ ] Proofread for errors
