## Critical Risk Response (24h)

### Immediate Actions
- [ ] Acknowledge customer concern
- [ ] Assign senior owner
- [ ] Escalate to leadership
- [ ] Prepare discount/credit options
- [ ] Schedule executive call if needed

### Investigation
- [ ] Review full customer history
- [ ] Understand root cause
- [ ] Identify if solvable
- [ ] Document findings
- [ ] Prepare response options

### Resolution
- [ ] Present solution options
- [ ] Negotiate terms if needed
- [ ] Get customer agreement
- [ ] Document outcome
- [ ] Schedule follow-up

---

## High Risk Response (48h)

### Assessment
- [ ] Review health score breakdown
- [ ] Identify primary risk factors
- [ ] Check recent interactions
- [ ] Assess relationship strength
- [ ] Evaluate save likelihood

### Outreach
- [ ] CSM personal outreach
- [ ] Discovery call scheduled
- [ ] Value reinforcement messaging
- [ ] Training/support offer
- [ ] Success story sharing

### Follow-Up
- [ ] Post-call action plan
- [ ] Regular check-ins scheduled
- [ ] Health score monitoring
- [ ] Document progress
- [ ] Escalate if no improvement

## Win-Back Program

### Eligibility Criteria
- [ ] Churned < 12 months ago
- [ ] Good standing at departure
- [ ] Solvable reason for churn
- [ ] No bad debt/fraud
- [ ] Contact info available

### Win-Back Sequence
- [ ] Day 30 post-churn outreach
- [ ] Day 60 value reminder
- [ ] Day 90 special offer
- [ ] Day 180 final attempt
- [ ] Track responses

### Win-Back Offers
- [ ] Discount on return
- [ ] Extended trial
- [ ] Free onboarding session
- [ ] Feature upgrade
- [ ] Flexible terms
