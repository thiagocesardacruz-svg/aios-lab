# Brand Consistency Checklist

> Ensures visual assets maintain brand identity

## Color Palette

- [ ] Primary colors are exact (hex values match)
- [ ] Secondary colors are used correctly
- [ ] Accent colors are applied sparingly
- [ ] Color ratios follow guidelines (60-30-10)
- [ ] Sufficient contrast for accessibility

## Typography

- [ ] Correct fonts are used
- [ ] Font weights match guidelines
- [ ] Font sizes follow hierarchy
- [ ] Line heights are consistent
- [ ] Letter spacing is correct

## Logo Usage

- [ ] Logo version is appropriate
- [ ] Clear space is maintained
- [ ] Minimum size is respected
- [ ] Logo is not distorted
- [ ] Background provides sufficient contrast

## Visual Elements

- [ ] Icon style is consistent
- [ ] Image style matches brand
- [ ] Illustration style is uniform
- [ ] Patterns/textures follow guidelines
- [ ] Photography direction is consistent

## Layout Patterns

- [ ] Grid system is followed
- [ ] Margins and padding are consistent
- [ ] Alignment follows brand rules
- [ ] White space is intentional
- [ ] Component styling is uniform

## Tone & Voice (Visual)

- [ ] Visual tone matches brand personality
- [ ] Energy level is appropriate
- [ ] Formality level is correct
- [ ] Emotion conveyed is on-brand
- [ ] Cultural sensitivity maintained

## Cross-Platform

- [ ] Consistent across all platforms
- [ ] Adapted appropriately per platform
- [ ] Recognition is maintained
- [ ] Core elements are present

## Documentation

- [ ] Follows brand guidelines document
- [ ] New elements are documented
- [ ] Exceptions are noted
- [ ] Source files are organized
