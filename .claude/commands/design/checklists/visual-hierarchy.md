# Visual Hierarchy Checklist

> Ensures proper information flow and attention guidance

## Primary Element

- [ ] Most important element is most prominent
- [ ] Uses size, color, or position for emphasis
- [ ] Immediately visible (< 1 second)
- [ ] Clear what action to take

## Information Flow

- [ ] Natural reading pattern (F or Z)
- [ ] Logical sequence of information
- [ ] Progressive disclosure of details
- [ ] No competing focal points

## Contrast & Emphasis

- [ ] Sufficient contrast between levels
- [ ] Important elements stand out
- [ ] Secondary elements are subdued
- [ ] Whitespace guides attention

## Typography Hierarchy

- [ ] Clear distinction: H1 > H2 > H3 > Body
- [ ] Size difference is noticeable
- [ ] Weight creates emphasis
- [ ] Color reinforces hierarchy

## Color Hierarchy

- [ ] Primary color on primary elements
- [ ] Accent color draws attention to CTA
- [ ] Neutral colors for supporting content
- [ ] Color guides the eye path

## Spacing Hierarchy

- [ ] Related items grouped together
- [ ] Unrelated items separated
- [ ] Consistent spacing within groups
- [ ] Larger gaps between sections

## Mobile Hierarchy

- [ ] Hierarchy works on small screens
- [ ] Touch targets are prioritized
- [ ] Scrolling reveals information logically
- [ ] Key info above fold

## Retention (Video/Motion)

- [ ] Hook in first 3 seconds
- [ ] Scene changes maintain interest
- [ ] Visual variety every 3-5 seconds
- [ ] CTA is clearly timed
- [ ] Ending drives action

## Accessibility

- [ ] Hierarchy works without color
- [ ] Screen readers can follow order
- [ ] Focus order is logical
- [ ] Landmarks are present
