# Creative Conversion Checklist

> Quality gate for conversion-focused creatives

## Visual Hierarchy

- [ ] Main message is instantly clear (< 3 seconds)
- [ ] Eye flow follows intended path
- [ ] Most important element has most contrast
- [ ] Secondary elements support, don't compete

## Call to Action

- [ ] CTA is visually prominent
- [ ] CTA color contrasts with background
- [ ] Action verb is clear (Get, Start, Join, etc.)
- [ ] CTA is above fold / early in sequence
- [ ] Only one primary CTA per creative

## Hook & Attention

- [ ] First element stops the scroll
- [ ] Unexpected or contrasting element present
- [ ] Curiosity is created
- [ ] Relevant to target audience

## Mobile Optimization

- [ ] Readable on mobile (no tiny text)
- [ ] Thumb-friendly tap targets
- [ ] Correct aspect ratio for platform
- [ ] Fast loading (optimized file size)

## Brand Alignment

- [ ] Colors match brand palette
- [ ] Typography is consistent
- [ ] Logo usage is correct
- [ ] Tone matches brand voice

## Platform Specs

- [ ] Correct dimensions for platform
- [ ] Text within safe zones
- [ ] File format is correct
- [ ] File size is optimized

## Trust & Credibility

- [ ] Social proof visible (if applicable)
- [ ] No misleading claims
- [ ] Professional quality
- [ ] Consistent with landing page

## Technical Quality

- [ ] No pixelation or blur
- [ ] Colors are accurate
- [ ] Text is sharp
- [ ] Proper file export settings
