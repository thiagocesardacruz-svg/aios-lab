# Content Production Checklist

Use este checklist para validar o pipeline de producao de conteudo.

## Metadata
```yaml
id: content-production-checklist
name: Checklist de Producao de Conteudo
version: 1.0.0
executor: content-operations-manager
```

---

## Pre-Producao
- [ ] Pauta definida e aprovada
- [ ] Formato definido (video, post, carrossel, story, live)
- [ ] Canal de publicacao definido (YouTube, Instagram, TikTok, etc.)
- [ ] Data de publicacao agendada
- [ ] Briefing criado no ClickUp
- [ ] Responsaveis atribuidos (roteiro, gravacao, edicao, design)
- [ ] Materiais de referencia reunidos

## Roteiro / Copy
- [ ] Roteiro escrito e revisado
- [ ] Hook/abertura forte
- [ ] CTA definido
- [ ] Palavras-chave incluidas (SEO)
- [ ] Tom de voz da Natalia mantido
- [ ] Aprovacao do roteiro pela Natalia

## Producao
- [ ] Gravacao realizada
- [ ] Qualidade de audio OK
- [ ] Qualidade de video OK
- [ ] Iluminacao adequada
- [ ] Enquadramento correto
- [ ] B-roll capturado (se aplicavel)

## Pos-Producao
- [ ] Edicao concluida
- [ ] Thumbnails criadas (3 opcoes)
- [ ] Titulo otimizado (SEO + CTR)
- [ ] Descricao escrita
- [ ] Tags definidas
- [ ] Legendas revisadas
- [ ] Cards/end screens configurados (YouTube)

## Publicacao
- [ ] Upload realizado
- [ ] Agendamento confirmado
- [ ] Redes sociais de distribuicao preparadas
- [ ] Notificacao da equipe

## Pos-Publicacao
- [ ] Metricas de 24h verificadas
- [ ] Comentarios respondidos
- [ ] Clips/reels derivados criados
- [ ] Task atualizada no ClickUp para "Publicado"
